# 台灣資通安全法規詳解

## 資通安全管理法

### 立法歷程
- **2018年6月6日**: 總統公布
- **2019年1月1日**: 正式施行
- **2025年8月29日**: 立法院三讀通過修正案
- **2025年9月24日**: 總統公布修正案
- **2025年12月1日**: 修正案實施

### 修法背景
資通安全管理法自2019年實施以來，為我國首部資通安全專法，確立國家資安治理的法律基礎。2025年修正案是該法實施以來的首次修訂，主要因應：
1. 政府組織改造（數位發展部成立）
2. 資安威脅日益嚴峻
3. 關鍵基礎設施防護需求提升
4. 國際資安合作需求

### 法律架構

#### 主管機關
- **修正前**: 行政院
- **修正後**: 數位發展部

#### 適用對象
1. **公務機關**:
   - 中央及地方各級機關
   - 公立學校
   - 公營事業機構

2. **特定非公務機關**:
   - 關鍵基礎設施提供者
   - 政府捐助之財團法人
   - 其他經主管機關指定者

3. **政府捐助之財團法人**

### 2025年修正重點

#### 1. 禁用危害產品條款（第7條修正）
**修正內容**:
- 公務機關禁止下載、安裝或使用危害國家資通安全之產品
- 由建議性質提升至強制性法律義務
- 違反者可處分相關人員

**影響**:
- 政府採購需審慎評估資安風險
- 建立產品資安審查機制
- 可能影響特定國家或廠商產品

#### 2. 稽核權擴大（第10條修正）
**修正內容**:
- 數位發展部可稽核公務機關資安維護計畫實施情形
- 稽核範圍包含計畫制定、執行、成效

**影響**:
- 強化中央對地方的督導能力
- 提升整體政府資安水準
- 機關需更重視資安維護計畫執行

#### 3. 專責人力要求（第14條修正）
**修正內容**:
- 特定非公務機關須設置專責資安人員
- 須設置資安長（Chief Information Security Officer, CISO）
- 人數依機關規模與業務性質而定

**影響**:
- 提升民間關鍵單位資安能量
- 創造資安人才需求
- 增加企業營運成本

#### 4. 罰則加重（第18條、第21條修正）
**修正內容**:

| 違規項目 | 修正前 | 修正後 |
|---------|-------|-------|
| 未通報資安事件 | NT$30萬~300萬 | NT$100萬~1,000萬 |
| 違反維護計畫要求 | NT$20萬~200萬 | NT$50萬~500萬 |
| 未配合調查 | NT$20萬~200萬 | NT$50萬~500萬 |

**影響**:
- 強化嚇阻效果
- 提升組織對資安合規的重視
- 違規成本大幅增加

#### 5. 調查權力（新增第15-1條）
**修正內容**:
- 中央目的事業主管機關可調查特定非公務機關重大資安事件
- 調查範圍包含事件原因、影響、改善措施
- 被調查單位有配合義務

**影響**:
- 強化政府對重大資安事件的掌握能力
- 提升事件應變效率
- 促進跨機關合作

### 實務應用

#### 公務機關應辦事項

**A級機關**:
- 設置資安專責單位
- 配置專職資安人員不少於3人
- 每年辦理資安健診
- 每年辦理演練
- 設置資安長（簡任或相當職務以上）

**B級機關**:
- 設置資安專責單位或指定專人
- 配置專職資安人員不少於1人
- 每2年辦理資安健診
- 每年辦理演練

**C級機關**:
- 指定資安專責人員
- 每3年辦理資安健診
- 每2年辦理演練

**D級、E級機關**:
- 指定資安聯絡人
- 依需要辦理資安健診與演練

#### 特定非公務機關應辦事項
1. 提報資安維護計畫
2. 設置資安長及專責人員
3. 定期辦理資安教育訓練
4. 依規定通報資安事件
5. 配合中央目的事業主管機關稽核
6. 參與資安情資分享機制

#### 資安事件通報時限

| 事件等級 | 通報時限 | 範例 |
|---------|---------|------|
| 第1級（緊急） | 1小時內 | 重大服務中斷、大量個資外洩 |
| 第2級（重大） | 3小時內 | 系統遭入侵、勒索軟體感染 |
| 第3級（一般） | 1日內 | 惡意程式感染、異常連線 |
| 第4級（注意） | 3日內 | 可疑掃描、輕微異常 |

### 罰則彙整

| 違規行為 | 法條 | 罰則 |
|---------|-----|------|
| 未依規定訂定或實施維護計畫 | 第18條 | NT$50萬~500萬，並限期改善，屆期未改善按次處罰 |
| 未依規定通報資安事件 | 第21條 | NT$100萬~1,000萬 |
| 未依規定配合調查 | 第21條 | NT$50萬~500萬，並得按次處罰 |
| 未依規定辦理資安稽核 | 第18條 | NT$50萬~500萬 |
| 違反禁用產品規定 | 行政處分 | 視情節輕重處分相關人員 |

## 資通安全責任等級分級辦法

### 分級原則

#### A級機關
**定義**: 對國家安全、社會秩序或公共利益有重大影響
**範例**:
- 總統府
- 行政院及所屬一級機關
- 國防部及所屬機關
- 中央銀行
- 關鍵基礎設施主管機關

#### B級機關
**定義**: 對國家安全、社會秩序或公共利益有顯著影響
**範例**:
- 中央二級機關
- 直轄市政府
- 重要公營事業
- 主要國立大學

#### C級機關
**定義**: 對國家安全、社會秩序或公共利益有一定影響
**範例**:
- 縣市政府
- 地方公營事業
- 一般國立大學
- 中小型中央機關

#### D級機關
**定義**: 對國家安全、社會秩序或公共利益影響較小
**範例**:
- 鄉鎮市區公所
- 小型地方機關

#### E級機關
**定義**: 對國家安全、社會秩序或公共利益影響輕微
**範例**:
- 極小型地方機關
- 特定公立學校

### 分級考量因素
1. 機關層級與職掌
2. 業務性質與重要性
3. 資通系統規模與複雜度
4. 保有資料之機敏程度
5. 服務對象與範圍
6. 中斷服務之影響程度

### 各級應辦事項差異

| 項目 | A級 | B級 | C級 | D級 | E級 |
|-----|-----|-----|-----|-----|-----|
| 專責單位 | 必須設置 | 必須設置或專人 | 指定專人 | 指定聯絡人 | 指定聯絡人 |
| 專職人員 | ≥3人 | ≥1人 | - | - | - |
| 資安長 | 簡任以上 | 薦任以上 | - | - | - |
| 資安健診 | 每年 | 每2年 | 每3年 | 依需要 | 依需要 |
| 演練頻率 | 每年 | 每年 | 每2年 | 依需要 | 依需要 |
| 滲透測試 | 每年 | 每2年 | 依需要 | 依需要 | 依需要 |

## 資通安全事件通報及應變辦法

### 事件分級

#### 第1級（緊急）
**定義**: 對國家安全、社會秩序或公共利益造成重大危害
**範例**:
- 核心系統遭入侵且服務中斷
- 10萬筆以上個資外洩
- 關鍵基礎設施遭受攻擊

**通報時限**: 1小時內

#### 第2級（重大）
**定義**: 對國家安全、社會秩序或公共利益造成顯著危害
**範例**:
- 重要系統遭入侵
- 1萬~10萬筆個資外洩
- 勒索軟體感染多台設備

**通報時限**: 3小時內

#### 第3級（一般）
**定義**: 對國家安全、社會秩序或公共利益造成一定危害
**範例**:
- 一般系統遭入侵
- 1,000~1萬筆個資外洩
- 惡意程式感染少量設備

**通報時限**: 1日內

#### 第4級（注意）
**定義**: 對國家安全、社會秩序或公共利益影響輕微
**範例**:
- 異常連線活動
- 可疑掃描行為
- 小範圍異常

**通報時限**: 3日內

### 通報流程

#### 公務機關
1. 發現事件 → 初判等級
2. 通報 NCERT (國家資通安全通報應變網站)
3. NCERT 協調處理
4. 事件應變與調查
5. 恢復與復原
6. 事後檢討報告

#### 特定非公務機關
1. 發現事件 → 初判等級
2. 通報中央目的事業主管機關
3. 副知 TWCERT/CC
4. 事件應變與調查
5. 恢復與復原
6. 事後檢討報告

### 應變機制

#### 中央應變組織
- **指揮官**: 數位發展部部長
- **執行單位**: 資通安全署
- **技術支援**: 國家資通安全研究院

#### 應變流程
1. **啟動**: 接獲重大事件通報
2. **評估**: 判定影響範圍與嚴重程度
3. **協調**: 召集相關機關協調應變
4. **支援**: 提供技術與資源支援
5. **監控**: 持續監控事件發展
6. **結案**: 確認事件解決

## 常見問答 (FAQ)

### Q1: 我的組織需要遵守資通安全管理法嗎？
**A**: 如果您的組織是：
- 政府機關
- 公立學校
- 公營事業
- 關鍵基礎設施提供者
- 政府捐助的財團法人
- 經主管機關指定的單位

則需遵守該法。

### Q2: 如何確認我的機關責任等級？
**A**: 由中央目的事業主管機關依《資通安全責任等級分級辦法》核定，考量：
- 機關層級與職掌
- 業務性質與重要性
- 資通系統規模
- 資料機敏程度

### Q3: 發生資安事件該如何通報？
**A**:
- **公務機關**: 通報 NCERT (https://www.ncert.nat.gov.tw/)
- **企業**: 通報 TWCERT/CC (https://www.twcert.org.tw/)
- 依事件等級於規定時限內通報

### Q4: 未通報資安事件的後果？
**A**: 依資通安全管理法第21條，可處 NT$100萬~1,000萬罰鍰

### Q5: 需要設置多少資安人員？
**A**: 依責任等級：
- A級: ≥3人專職
- B級: ≥1人專職
- C級以下: 視需要配置

### Q6: 資安健診頻率為何？
**A**:
- A級: 每年
- B級: 每2年
- C級: 每3年
- D、E級: 依需要辦理

### Q7: 哪些產品不能使用？
**A**: 經主管機關公告「危害國家資通安全之產品」，公務機關不得使用。具體清單由資通安全署公告。

### Q8: 中小企業如何強化資安？
**A**: 可申請：
- NICS 台灣資安計畫（免費諮詢輔導）
- 經濟部資安輔導計畫
- 使用基礎防護工具（防火牆、端點防護、備份）
