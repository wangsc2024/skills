# 台灣資安訓練與學習資源

## 官方訓練計畫

### AIS3 (Advanced Information Security Summer School)

#### AIS3 Junior (高中生資安研習營)
- **對象**: 高中職學生
- **時間**: 每年暑假
- **天數**: 5天4夜
- **名額**: 約100-150人
- **費用**: 免費（含食宿）
- **報名**: 需通過書面審查

**課程內容**:
- 資安基礎觀念
- CTF 入門
- Web 安全基礎
- Binary 基礎
- Reverse Engineering 入門
- Cryptography 基礎
- 實作練習

**特色**:
- 業界講師授課
- 小組競賽
- 交流互動
- 培養資安興趣

#### AIS3 Main Program (進階資安暑期課程)
- **對象**: 大學生、研究生
- **時間**: 每年暑假
- **天數**: 7-10天
- **名額**: 約50-80人
- **費用**: 部分補助
- **報名**: 需通過書面審查及技術測驗

**課程領域**:
- **Pwn (Binary Exploitation)**:
  - Stack overflow
  - Heap exploitation
  - ROP (Return-Oriented Programming)
  - Kernel exploitation

- **Web Security**:
  - SQL Injection
  - XSS (Cross-Site Scripting)
  - SSRF (Server-Side Request Forgery)
  - Deserialization attacks
  - Web framework vulnerabilities

- **Reverse Engineering**:
  - Static analysis
  - Dynamic analysis
  - Malware analysis
  - Anti-debugging techniques

- **Cryptography**:
  - Classical ciphers
  - Modern cryptography
  - Cryptanalysis
  - Applied cryptography

- **Forensics**:
  - Disk forensics
  - Memory forensics
  - Network forensics
  - Mobile forensics

**特色**:
- 頂尖業界講師
- 實戰演練
- CTF 競賽
- 職涯指引
- 業界交流

#### AIS3 EOF (End of Fun)
- **性質**: AIS3 Pre-exam 後的線上 CTF
- **參與**: AIS3 Pre-exam 參賽者
- **形式**: Jeopardy 式 CTF
- **時間**: 48小時

### 職涯建議
1. **打好基礎**: 作業系統、網路、程式設計
2. **選擇方向**: 防禦、攻擊、管理、研究
3. **實戰經驗**: CTF、實習、專案
4. **持續學習**: 技術日新月異
5. **建立人脈**: 參與社群、會議
6. **取得認證**: 提升競爭力（非必要但加分）
7. **培養軟實力**: 溝通、簡報、團隊合作

## 學習路徑建議

### 初學者 (0-1年)
1. **基礎知識**:
   - Linux 基礎指令
   - 網路基礎 (TCP/IP, HTTP)
   - 程式設計 (Python, C)

2. **入門課程**:
   - AIS3 Junior (如果是高中生)
   - MyFirstCTF
   - PortSwigger Web Security Academy

3. **練習平台**:
   - picoCTF
   - TryHackMe (Easy rooms)

### 中階 (1-2年)
1. **專精領域選擇**:
   - Web, Pwn, Reverse, Crypto 選一兩項

2. **進階課程**:
   - AIS3 Main
   - SCIST 課程
   - 線上專業課程

3. **競賽參與**:
   - AIS3 Pre-exam
   - 台灣本土 CTF
   - 國際 CTF (Beginner 級)

4. **練習平台**:
   - pwnable.tw
   - HackTheBox (Medium machines)

### 進階 (2年以上)
1. **深度技術研究**:
   - 0-day 研究
   - 漏洞挖掘
   - Exploit 開發

2. **國際競賽**:
   - HITCON CTF
   - Balsn CTF
   - 國際頂尖 CTF

3. **業界實習/工作**:
   - 資安公司
   - Bug Bounty
   - 研究職位

4. **社群貢獻**:
   - 技術分享
   - 開源專案
   - Writeup 撰寫
