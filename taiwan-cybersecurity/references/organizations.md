# 台灣資通安全組織機構詳解

## 政府機構

### 數位發展部 (Ministry of Digital Affairs, MODA)

#### 基本資訊
- **成立時間**: 2022年8月27日
- **前身**: 行政院科技會報辦公室、行政院資通安全處部分業務
- **部長**: 黃彥男（截至2026年）
- **地址**: 台北市中正區延平南路143號
- **電話**: 02-2380-8500
- **官網**: https://moda.gov.tw/

#### 組織架構
- 綜合規劃司
- 資源管理司
- 數位產業署
- **資通安全署** (重點業務)
- 民主網絡司
- 多元創新司

#### 核心任務
1. 數位國家政策規劃與執行
2. 資通安全政策與管理
3. 數位產業發展
4. 數位基礎建設
5. 數位人權與治理

### 資通安全署 (Administration for Cyber Security, ACS)

#### 基本資訊
- **成立時間**: 2022年8月（隨數位發展部成立）
- **前身**: 行政院國家資通安全會報技術服務中心
- **署長**: [依最新公告]
- **地址**: 台北市中正區延平南路143號
- **電話**: 02-2380-8500
- **官網**: https://moda.gov.tw/ACS/

#### 核心業務
1. **資安政策規劃**:
   - 國家資通安全政策制定
   - 資通安全管理法及子法制修訂
   - 資安標準與指引制定

2. **關鍵基礎設施防護**:
   - 關鍵資訊基礎設施識別與指定
   - 行業別資安防護標準制定
   - 關鍵基礎設施稽核

3. **資安事件通報應變**:
   - 24/7 資安事件通報受理
   - 跨機關應變協調
   - 重大事件調查

4. **資安稽核與查核**:
   - 公務機關資安稽核
   - 特定非公務機關查核
   - 資安成熟度評估

5. **資安產業發展**:
   - 資安產業政策推動
   - 資安新創輔導
   - 國際資安合作

#### 業務專區
- 資安法規專區
- 關鍵基礎設施資安防護
- 資安事件通報應變
- 資安人才培育
- 資安產業發展

#### 重要計畫
- 國家資通安全發展方案
- 關鍵資訊基礎設施防護計畫
- 資安產業發展行動計畫

### 國家資通安全研究院 (NICS)

#### 基本資訊
- **全名**: 國家資通安全研究院
- **英文**: National Institute of Cyber Security
- **性質**: 行政法人
- **成立時間**: 2023年1月1日
- **監督機關**: 數位發展部
- **院長**: [依最新公告]
- **地址**: [依官網公告]
- **電話**: 02-6631-1881
- **官網**: https://www.nics.nat.gov.tw/
- **GitHub**: https://github.com/nics-tw

#### 設立宗旨
依據《國家資通安全研究院設置條例》，以研發資通安全技術、應用及移轉，協助政府機關及關鍵基礎設施提供者強化資通安全防護能力，並培育資通安全專業人才為宗旨。

#### 組織架構
1. **前瞻研究中心**
   - 架構設計組
   - 技術研發組
   - 數位韌性研究

2. **資安防護中心**
   - 政府組態基準 (GCB) 維運
   - 共通規範制定
   - 防護技術支援

3. **國際合作及資安治理中心**
   - 國際資安合作
   - 資安政策研究
   - 法制觀測

4. **人才培育中心**
   - 資安課程規劃
   - 認證制度
   - 訓練計畫執行

5. **TWCERT/CC 營運**
   - 企業資安通報應變
   - 威脅情資分享
   - 漏洞通報協調

#### 核心業務

##### 1. 技術研發
- 資安防護技術
- 威脅偵測與分析
- 資安檢測工具
- 數位鑑識技術

##### 2. 政府組態基準 (GCB)
**涵蓋系統**:
- Windows Server 系列
- Windows 10/11
- Linux (RHEL, Ubuntu, CentOS)
- Exchange Server
- SQL Server
- PostgreSQL
- MySQL
- Apache Web Server
- IIS
- 網路設備（防火牆、交換器）

**文件格式**:
- 說明文件 (PDF)
- 檢測腳本
- 設定範本

**官網**: https://www.nics.nat.gov.tw/core_business/cybersecurity_defense/GCB/

##### 3. 共通規範
- 政策面控制措施
- 管理面控制措施
- 技術面控制措施
- 參考國際標準（ISO 27001、NIST CSF）

**官網**: https://www.nics.nat.gov.tw/cybersecurity_resources/reference_guide/Common_Standards/

##### 4. 資通安全弱點通報系統 (VANS)
- **全名**: Vulnerability Analysis and Notice System
- **功能**:
  - 產品漏洞通報
  - 漏洞分析與評估
  - 協調廠商修補
  - 漏洞資訊發布
- **官網**: https://vans.nat.gov.tw/

##### 5. NICS 台灣資安計畫
**服務對象**:
- 中小型企業
- 微型企業
- 非營利組織

**服務內容**:
- 免費資安諮詢
- 現場輔導
- 資安健檢
- 教育訓練
- 資源媒合

**申請方式**: 透過官網線上申請

##### 6. 數位韌性教材
**三大面向**:

1. **高可用性 (High Availability)**
   - 資料中心營運知識
   - 電力與冷卻管理
   - DNS 與 CDN 設定
   - 網路韌性設計

2. **可維護性 (Maintainability)**
   - 原始碼獨立驗證流程
   - 漏洞通報自動化
   - CI/CD 安全實踐
   - 元件掃描工具使用

3. **易用性 (Usability)**
   - 隱私權政策設計
   - 無廣告服務嵌入
   - 使用者體驗安全

**GitHub**: https://github.com/nics-tw/resilience-material

##### 7. 國際資安政策法制觀測
- 每週發布國際資安政策法制觀測週報
- 追蹤美、歐、日、韓等國資安政策
- 分析國際資安趨勢
- 提供政策建議

#### 出版品
1. 資通安全法規彙編
2. 政府組態基準系列文件
3. 國際資安政策法制觀測週報
4. 資安技術白皮書
5. 網路攻防演練報告

#### 國際合作

### TWCERT/CC (台灣電腦網路危機處理暨協調中心)

#### 基本資訊
- **全名**: Taiwan Computer Emergency Response Team / Coordination Center
- **營運**: 國家資通安全研究院（2024年1月起）
- **前身**: 財團法人台灣網路資訊中心 (TWNIC) 營運（2001-2023）
- **服務對象**: 民間企業
- **官網**: https://www.twcert.org.tw/
- **服務時間**: 24/7 全天候

#### 核心功能

##### 1. 資安事件通報與應變
- 24/7 資安事件通報受理
- 事件分析與研判
- 應變建議提供
- 跨組織協調
- 技術支援

**通報方式**:

##### 2. 威脅情資分享
