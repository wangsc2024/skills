# 公文用語詳解

本文件詳細說明公文各類用語的正確使用方式。

---

## 一、附送語

### 用語區分

| 用語 | 適用對象 | 說明 |
|------|----------|------|
| 檢陳、附陳 | 上級機關 | 對上級送附件時使用 |
| 檢送 | 平行/下級機關 | 送主要文件時使用 |
| 檢附 | 平行/下級機關 | 送附帶文件時使用 |
| 檢同 | 平行/下級機關 | 附件與公文同等重要時 |
| 檢具 | 平行/下級機關 | 出具證明文件時 |
| 檢還 | 平行/下級機關 | 歸還原文件時 |
| 附、附送 | 平行/下級機關 | 一般附件 |

### 主旨範例

**對上級機關**：
```
主旨：檢陳本校○○○年度校務發展計畫1份，請鑒核。
```

**對平行機關**：
```
主旨：檢送「○○○研討會」相關資料1份，請查照。
```

**對下級機關**：
```
主旨：檢附「○○辦法」修正條文1份，請轉知所屬，請查照。
```

## 二、正本與副本

### 基本規則

| 項目 | 說明 |
|------|------|
| 正本 | 主要受文對象，需蓋印或電子簽章 |
| 副本 | 知會對象，蓋印與正本同 |
| 抄本 | 機關內部使用，不需蓋印 |

### 副本使用時機

1. **知會相關機關**：通案分行之相關機關
2. **查詢案件**：將查詢函副本抄送來文機關
3. **越級行文**：緊急情況副本抄送直屬上下級機關
4. **配合辦理**：要求副本收受者協辦

### 副本附件處理

- 原則上附件**僅隨附於正本**
- 副本需附件時，於機關名稱後註明「(含附件)」
- 要求副本收受者配合辦理時，在說明段敘明

### 範例

**正本有附件，副本不需附件**：
```
正本：教育部
副本：本校各單位
附件：計畫書1份
```

**副本也需附件**：
```
正本：教育部
副本：本校各單位（含附件）
附件：計畫書1份
```

**要求副本收受者配合**：
```
說明：
  ...
  三、副本抄送本府各機關，請配合辦理。

正本：行政院
副本：本府各機關
```

## 三、稱謂語詳解

### 機關稱謂

| 稱謂 | 適用情況 | 範例 |
|------|----------|------|
| **鈞** | 下級對上級（有隸屬關係） | 鈞部、鈞府、鈞院、鈞會 |
| **大** | 對上級（無隸屬關係） | 大院、大部 |
| **貴** | 上級對下級、平行機關 | 貴局、貴處、貴校、貴公司 |
| **本** | 自稱 | 本部、本府、本校、本局 |

### 人員稱謂

| 稱謂 | 適用情況 | 範例 |
|------|----------|------|
| **臺端** | 機關對屬員或人民 | 臺端所請事項... |
| **先生/女士** | 對人民（已知性別） | ○○○先生 |
| **君** | 對人民（不知性別或對象眾多） | ○○○君 |
| **鈞長/鈞座** | 屬員對長官 | 敬陳鈞長 |

### 使用範例

**下級對上級（有隸屬關係）**：
```
主旨：有關本市申請○○補助一案，請鈞府核示。
```

**對平行機關**：
```
主旨：有關貴局○○○一案，復如說明，請查照。
```

**對人民**：
```
主旨：臺端申請○○一案，經審核符合規定，請查照。
```

## 四、引敘語詳解

### 引述來文

| 用語 | 適用對象 | 範例 |
|------|----------|------|
| 奉 | 上級機關 | 奉鈞部○年○月○日○字第○號函 |
| 准 | 平行機關 | 准貴局○年○月○日○字第○號函 |
| 據 | 下級機關/人民 | 據○○○○年○月○日申請書 |

### 引述發文後續函

| 用語 | 適用對象 |
|------|----------|
| 諒蒙鈞察 | 對上級機關發文後續函 |
| 諒達 | 對平行機關發文後續函 |
| 計達 | 對下級機關發文後續函 |

### 引述大意

| 用語 | 說明 |
|------|------|
| 略以、略謂 | 引述來文大意時使用 |
| 略稱 | 引述來文大意時使用 |

### 範例

**引述上級來函**：
```
說明：
  一、奉行政院○○○年○月○日院臺○字第○○○號函辦理。
```

**引述平行機關來函**：
```
說明：
  一、准貴局○○○年○月○日○字第○○○號函辦理。
```

**引述人民申請**：
```
說明：
  一、據臺端○○○年○月○日申請書辦理。
```

**引述來文大意**：
```
說明：
  一、依據○○部○年○月○日函略以，各機關應加強...
```

## 五、期望語詳解

### 上行文期望語

| 用語 | 適用情況 |
|------|----------|
| 請鑒核 | 報核案件 |
| 請核示 | 請示案件 |
| 請鑒察 | 僅供瞭解 |
| 請核備 | 報備案件 |
| 敬請鈞長核示 | 對長官 |

### 平行文期望語

| 用語 | 適用情況 |
|------|----------|
| 請查照 | 一般通知 |
| 請惠復 | 請求答復 |
| 請同意惠復 | 請求同意 |
| 請惠予同意見復 | 請求同意並復 |
| 請查照辦理見復 | 請辦理並復 |

### 下行文期望語

| 用語 | 適用情況 |
|------|----------|
| 請照辦 | 要求辦理 |
| 希查照 | 通知周知 |
| 請查照 | 一般通知 |
| 請轉行照辦 | 轉知辦理 |

### 請示語（期望語前）

| 用語 | 範例 |
|------|------|
| 是否可行 | 是否可行，請核示 |
| 當否 | 當否，請核示 |
| 是否有當 | 是否有當，請鑒核 |
| 可否之處 | 可否之處，請核示 |

**注意**：請示語後用「，」不用「？」

## 六、准駁語詳解

### 許可用語

| 用語 | 適用情況 |
|------|----------|
| 應予照准 | 上級對下級 |
| 准予照辦 | 上級對下級 |
| 准予備查 | 報備案件 |
| 同意照辦 | 平行機關 |
| 敬表同意 | 平行機關 |
| 符合規定 | 依法准許 |

### 不許可用語

| 用語 | 適用情況 |
|------|----------|
| 未便照准 | 一般不准 |
| 礙難照准 | 有困難不准 |
| 應從緩議 | 暫緩考慮 |
| 應予駁回 | 明確駁回 |
| 歉難同意 | 婉拒 |
| 與規定不符 | 依法不准 |

### 應避免用語

| 不當用語 | 正確用語 |
|----------|----------|
| 似無不可 | 應予照准 |
| 尚無不合 | 符合規定 |
| 似可照准 | 准予照辦 |
| 應該可以 | 同意照辦 |
| 不盡相同 | 與規定不符 |

## 七、經辦語詳解

### 用語對照

| 用語 | 適用對象 |
|------|----------|
| 遵經、遵即 | 對上級機關 |
| 業經、經已 | 通用 |
| 均經 | 通用 |
| 迭經 | 多次經過 |
| 旋經 | 隨即經過 |
| 嗣經 | 其後經過 |

### 範例

```
說明：
  二、遵經本府召開會議研商，決議如下：...
```

```
說明：
  二、本案業經本校○月○日○○會議審議通過。
```
