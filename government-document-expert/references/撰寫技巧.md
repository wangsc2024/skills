# 公文撰寫技巧詳解

本文件提供公文各段落的詳細撰寫技巧與常見錯誤修正。

---

## 第一部分：主旨段撰寫技巧

### 主旨段公式
```
主旨 = 起首語 + 本案主要意旨 + 期望語（≤60字）
```

### 常用起首語對照表

| 用途 | 起首語範例 |
|------|-----------|
| 檢送資料 | 檢送、檢陳（對上級）、檢附 |
| 說明事由 | 有關、關於、為、茲、查 |
| 請求核准 | 擬請、謹請、敬請 |
| 邀請出席 | 敬邀、函請、邀請 |
| 轉知周知 | 轉知、函轉、轉頒 |
| 同意事項 | 同意、准予、應予照准 |
| 答復來函 | 復貴○○函、有關○○一案 |

### 期望語對照表

| 行文類型 | 期望語 | 用途說明 |
|----------|--------|----------|
| **上行文** | 請鑒核 | 報核案件 |
| | 請核示 | 請示案件 |
| | 請鑒察 | 僅供瞭解 |
| | 請核備 | 報備案件 |
| **平行文** | 請查照 | 一般通知 |
| | 請惠復 | 請求答復 |
| | 請同意惠復 | 請求同意 |
| | 請查照辦理見復 | 請辦並復 |
| **下行文** | 請照辦 | 要求辦理 |
| | 希查照 | 通知周知 |
| | 請轉行照辦 | 轉知辦理 |

### 主旨段撰寫要點

1. **字數限制**：60字以內（建議50字）
2. **不可分項**：一段完成，不可「一、二、三」
3. **文字緊接**：主旨段名冒號後直接書寫
4. **語意完整**：包含案由＋目的或擬辦
5. **避免重複**：期望語不在說明段重複

### 主旨段範例

**正確範例**：
```
主旨：檢送本校○○○年度校務發展計畫1份，請鑒核。
```

```
主旨：有關本縣申請○○補助經費一案，業獲核定，請查照。
```

**錯誤範例**：
```
❌ 主旨：一、檢送計畫。二、請核示。
   （錯誤：主旨分項）

❌ 主旨：為辦理本案業務，依據相關規定，經審慎評估，
        本府認為應予同意，爰擬具相關方案，簽請核示。
   （錯誤：超過60字，過於冗長）
```

## 第二部分：說明段撰寫技巧

### 說明段結構公式
```
說明 = 引據 + 申述 + 歸結 + 補充（分項條列）
```

### 各要素撰寫要點

#### 一、引據（第一項）
**定義**：辦理本案的原因與依據

**可引據的內容**：
- 來函：依據○○○年○月○日○字第○號函辦理
- 法令：依據○○法第○條規定
- 計畫：依據「○○計畫」辦理
- 會議決議：依據本府○月○日○○會議決議辦理
- 簽陳：依據本案前簽辦理
- 前案：案經本府○○○年○月○日○字第○號函核定
- 現況：本府轄區○○○情形...

**引據範例**：
```
一、依據教育部○○○年○月○日臺教高（一）字第○○○號函辦理。
```

```
一、依據「政府採購法」第22條第1項第2款規定辦理。
```

#### 二、申述（第二項起）
**定義**：說明案件的來龍去脈、分析利弊因素

**應包含內容**：
- 案件背景
- 處理經過
- 人事時地物
- 利弊分析
- 相關數據

**申述範例**：
```
二、本案緣起於○○○，經本府○月○日召開研商會議，
    決議○○○。
三、經評估，本案辦理效益如下：
  （一）可提升○○效率○○%。
  （二）預估節省經費○萬元。
```

#### 三、歸結（最後一項前）
**定義**：提出具體要求或作法

**歸結範例**：
```
四、本案擬函請各單位配合辦理。
```

#### 四、補充事項
**內容**：
- 副本收受者的配合事項
- 附件說明（置於最後一項）

**範例**：
```
五、副本抄陳○○部，請轉知所屬周知。
六、檢附相關資料1份。
```

### 論證方式

#### 推理式（引據→申述→歸結）
適用於：說明性、分析性公文

```
說明：
  一、依據○○辦理。（引據）
  二、本案經過○○○。（申述）
  三、綜上，擬○○○。（歸結）
```

#### 因果式（事實→原因→結果）
適用於：報告性公文

```
說明：
  一、本府轄區○○發生○○情形。（事實）
  二、經查原因為○○○。（原因）
  三、已採取○○措施，情況已獲控制。（結果）
```

## 第三部分：擬辦段撰寫技巧

### 擬辦段定義
提出具體處理意見或解決方案，供長官核定或選擇。

### 撰寫要點

1. **具體可行**：提出明確的處理建議
2. **分項條列**：多項建議時分項說明
3. **供選擇**：必要時提供甲乙方案
4. **結尾語**：「陳請核示」或「陳請鑒核」

### 擬辦範例

**單一建議**：
```
擬辦：
  一、擬同意辦理。
  二、陳請核示。
```

**多項建議**：
```
擬辦：
  一、本案擬同意補助。
  二、如蒙核可，擬函復申請單位。
  三、陳請核示。
```

**選擇方案**：
```
擬辦：
  甲案：同意全額補助。
  乙案：同意部分補助（○萬元）。
  以上二案，擬採甲案，陳請核示。
```

## 第四部分：常見錯誤與修正

### 錯誤類型一：格式問題

| 錯誤 | 修正 |
|------|------|
| 主旨分項 | 主旨不分項，一段完成 |
| 主旨超過60字 | 精簡文字或移至說明段 |
| 段名後空格 | 文字緊接冒號後書寫 |

### 錯誤類型二：用語問題

| 錯誤寫法 | 正確寫法 |
|----------|----------|
| 請 查照 | 請查照 |
| 請 鑒核 | 請鑒核 |
| 請 同意惠復 | 請同意惠復 |
| 如左 | 如下 |
| 左列 | 下列 |

### 錯誤類型三：語氣問題

| 不當語氣 | 正確語氣 |
|----------|----------|
| 似無不可 | 應予照准 |
| 尚無不合 | 符合規定 |
| 似可照准 | 准予照辦 |
| 應該可以 | 同意辦理 |

### 錯誤類型四：引用問題

**錯誤**：自行重組法令原文
```
❌ 依據政府採購法規定，機關辦理採購，其金額達公告金額
   以上時，應公開招標。
```

**正確**：照錄原文並加引號
```
✅ 依據「政府採購法」第19條規定：「機關辦理公告金額
   以上之採購，除依第20條及第22條辦理者外，應公開
   招標。」
```

**引述大意時**：
```
✅ 依據「政府採購法」第19條規定略以，機關辦理公告金額
   以上之採購，除依規定情形外，應公開招標。
```

### 錯誤類型五：前後不一致

| 錯誤 | 說明 |
|------|------|
| 前稱「張先生」，後改「張君」 | 同一公文內稱謂應一致 |
| 前稱「格式及內容」，後改「內容及格式」 | 詞序應前後一致 |
| 機關名稱時全時簡 | 首次全稱並註明簡稱，後用簡稱 |

## 第五部分：撰寫檢核清單

### 主旨段檢核
- [ ] 字數≤60字
- [ ] 未分項
- [ ] 包含起首語
- [ ] 包含期望語
- [ ] 期望語與行文對象相符

### 說明段檢核
- [ ] 第一項為引據
- [ ] 引據格式正確（機關、日期、字號）
- [ ] 申述邏輯清晰
- [ ] 附件置於最後一項
- [ ] 未重複主旨的期望語

### 擬辦段檢核
- [ ] 有具體建議
- [ ] 建議可行
- [ ] 結尾有「陳請核示」

### 整體檢核
- [ ] 用語前後一致
- [ ] 無不當空格
- [ ] 引用格式正確
- [ ] 無雙重否定語氣
- [ ] 橫式書寫用「如下」非「如左」
