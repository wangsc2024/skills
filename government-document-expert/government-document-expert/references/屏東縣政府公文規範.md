## 三、屏東縣政府公文管理實施要點

### 法規依據
- 發布日期：中華民國104年8月18日
- 發布機關：屏東縣政府

### 公文分類管制

#### 一般公文

| 速別 | 定義 | 處理時限 |
|------|------|----------|
| 最速件 | 最緊急需立即處理之案件 | 1日內 |
| 速件 | 次於最速件亦應從速處理完畢之案件 | 3日內 |
| 普通件 | 一般案件 | 6日內 |

**計算方式**：以工作天計算，扣除假日

#### 人民申請案件

- 以「案」為單元管制
- 依相關法令規定之期限辦理
- 含假日（日曆天）計算

#### 人民陳情案件

- 依行政程序法規定
- 本府訂定處理期限為**6日**
- 以工作天計算，扣除假日

#### 訴願案件

| 文書類型 | 處理時限 |
|----------|----------|
| 訴願答辯書 | 20日內提出 |
| 訴願決定書 | 3個月內為之 |

#### 監察案件

實施全程時效管制

### 展期規定

1. 預計無法於時限內辦結，應於期限前簽請展期
2. 人民申請案件展期以一次為限
3. 展期日數不得超過原處理時限
4. 應於原期限屆滿前通知申請人

## 四、屏東縣政府文書用印作業須知

### 適用範圍

本須知適用於屏東縣政府及所屬機關。

### 文書定義

1. 機關與機關或機關與人民往來之公文書
2. 機關內部通行之文書
3. 公文以外與公務有關之文書

### 用印程序

```
承辦人撰擬文稿
    │
    ▼
單位主管核章
    │
    ▼
機關首長或授權主管簽章
    │
    ▼
蓋「代為決行」章（如適用）
    │
    ▼
送行政處文書檔案科（監印單位）
    │
    ▼
監印單位檢核
  ├─ 合格 → 蓋用印信
  └─ 不合格 → 退回補正
```

### 監印單位職責

- 監印單位：行政暨研考處文書檔案科
- 發現文書未經判行或有誤時，應退送補判或更正後再蓋印

### 特定文書簽陳要求

下列文書須先將內容與樣稿簽陳首長核可後方可用印：

| 文書類型 | 說明 |
|----------|------|
| 聘書 | 聘任人員使用 |
| 證明書 | 各類證明文件 |
| 契約書 | 採購、工程等契約 |
| 協議書 | 機關間或與民間協議 |
| 授權書 | 授權代理事項 |
| 獎狀 | 表揚獎勵用 |
| 執照 | 各類許可執照 |

### 文書製作注意事項

1. **機關全銜**：須繕有機關全銜
2. **紙張規格**：採用A4或A3紙張
3. **書寫方向**：直式橫書，由左至右
4. **蓋印位置**：預留蓋印位置應高低間隔適中，以免用印時重疊
5. **禁止塗改**：不得塗改或預留空白於用印後再自行補填
6. **日期填寫**：日期應詳填年月日

### 騎縫章規定

- 文書二頁以上需蓋騎縫章或職名章
- 用印順序：
  1. 相對人及承辦單位先行核章蓋騎縫章
  2. 監印單位檢核無誤後始蓋本府印信

### 契約修正規範

- 契約增刪修正應重新製作簽約
- 但小部分文字誤植經簽准後，可由相對人及承辦單位加蓋校對章

### 大宗定型文書

- 印製30張以上證書、聘書等需事先套印
- 應簽核監印單位，提供核准簽與樣稿
- 印模於15日內繳回

### 補發規定

- 補發文書需檢附申請表、補發原因及主管核章
- 監印單位應於補發文書加蓋補發章及日期

## 五、屏東縣政府電子公文線上簽核使用自然人憑證管理要點

### 法規依據
- 發文字號：屏府人考字第1010089010號函
- 發布日期：中華民國101年3月30日
- 生效日期：101年2月24日

### 立法目的

本府為辦理員工申請使用自然人憑證事項，推動電子公文線上簽核作業，特訂定本要點。

### 適用對象

本府負責公文承辦及簽核的人員。

### 自然人憑證定義

具有以下功能的憑證，由內政部憑證管理中心簽發：
- 身分識別
- 數位簽章
- 資料加解密

### 核發作業規定

| 時間 | 方式 |
|------|------|
| 101年3月7日前 | 由民政處及研考處集體申辦 |
| 101年3月8日後 | 申請人應親赴戶政事務所辦理 |

### 費用補助規定

| 情況 | 補助方式 |
|------|----------|
| 新證申領 | 全額補助 |
| 舊證到期 | 全額補助 |
| 憑證無法讀取 | 全額補助 |
| 憑證遺失 | 自行負擔 |
| 憑證損毀 | 自行負擔 |
| 姓名變更 | 自行負擔 |
| 身分證字號變更 | 自行負擔 |

申請補助須檢附：舊證、繳費收據及申請表

### 離職後效力

- 退休、離職或調職人員憑證無須繳回
- 可繼續使用至效期屆滿
- **但線上簽核識別功能自離職生效日失效**

- 公文系統客服電話：0800-222-188
- 使用前須安裝跨瀏覽器元件
- 需進行系統環境檢測

## 七、屏東縣法規標準自治條例

### 法規依據
- 公布日期：民國104年5月25日
- 最後修正：民國107年9月19日
- 主管單位：民政處

### 法規類型

| 類型 | 定義 | 命名規則 | 程序 |
|------|------|----------|------|
| 自治條例 | 經縣議會通過，由縣府公布 | 冠以「屏東縣」名稱 | 需經議會議決後報中央 |
| 自治規則 | 經縣府訂定並發布 | 規程、規則、細則、辦法、綱要、標準、準則 | 發布後函報中央備查 |
| 委辦規則 | 辦理上級機關委辦事項 | 依法定職權或授權訂定 | 依相關規定辦理 |

### 罰則限制

- 自治條例罰鍰上限：**新臺幣十萬元**
- 得規定按次處罰
- 可移送行政執行

### 公布程序

1. 自治條例需經議會議決後報中央主管機關
2. 自治規則應於發布後函報中央主管機關備查
3. **公告期間不得少於七日**

### 法規效力

- 條文應遵循憲法、法律及授權法規
- 牴觸者，無效

### 條文書寫格式

- 採分條橫式
- 分項、款、目三層級
- 可劃分編、章、節等結構
