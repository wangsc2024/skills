# 公文撰作案例集

本文件收錄各類型公文案例，依據公務人員保障暨培訓委員會《公文撰作解析》實作篇編排。

## 第一類：先簽後稿案例

### 案例1：向上層機關申請經費補助案

**情境**：市務會議中，市長指示環境景觀局向農委會爭取美化綠化補助經費，農委會函復同意補助5千萬元。

**簽辦方式選擇理由**：
- 金額重大（5千萬元）
- 需善用補助經費
- 需規劃執行方案
- 屬階段性公文

**簽**：
```
                先簽後稿
                    簽

主旨：有關鈞長指示申請美化綠化經費一案，業報經行政院
      農業委員會函復同意補助新臺幣5千萬元，謹擬具執
      行方案，簽請核示。

說明：
  一、依據行政院農業委員會○○○年○月○日農授水保字
      第○○○○○○○號函辦理。
  二、本案前奉鈞長於市務會議指示，向農委會爭取美化綠
      化補助經費，案經本局擬具計畫報經農委會審查，業
      獲同意補助。
  三、擬訂執行方案重點如下：
    （一）施作範圍：○○區○○路沿線。
    （二）施作期間：○○○年○月至○月。
    （三）預期效益：美化市容、提升生活品質。

擬辦：
  一、本案如蒙核可，擬即辦理後續發包作業。
  二、陳請鑒核。

                              敬陳  市長

承辦單位：環境景觀局    承辦人：○○○    分機：○○○○
```

### 案例2：邀請首長主持會議案

**情境**：工商協進會邀請院長與理事長共同主持「工商早餐會」。

**簽辦方式選擇理由**：
- 涉及首長行程
- 需先請示確定
- 核准後再發文答復

```
                先簽後稿
                    簽

主旨：中華民國工商協進會○理事長○○致院長箋函，為訂
      於本（000）年00月00日上午召開「工商早餐會」，
      邀請院長擔任共同主持人一案，簽請核示。

說明：
  一、依據工商協進會○理事長○○本年00月00日致院長箋
      函辦理。
  二、旨揭會議訂於本年00月00日（星期○）上午0時00分
      至0時00分假臺北○○飯店○○廳舉行。
  三、出席人員包括工商界代表約○○人。

擬辦：
  一、擬請院長出席主持。
  二、如蒙核可，擬復知工商協進會。
  三、陳請核示。

                              敬陳  院長

承辦單位：○○處    承辦人：○○○    分機：○○○○
```

## 第二類：簽稿併陳案例

### 案例4：轉知宣導案

**情境**：內政部函知加強防範一氧化碳中毒宣導。

**簽辦方式選擇理由**：
- 須說明案情背景
- 有時效性
- 需轉知所屬機關

**簽**：
```
                簽稿併陳
                    簽

主旨：內政部函知加強防範一氧化碳中毒宣導一案，擬函轉
      所屬機關配合辦理，簽請核示。

說明：
  一、依據內政部○○○年○月○日內授消字第○○○○號
      函辦理。
  二、近年因熱水器使用不當造成一氧化碳中毒案件頻傳，
      內政部請各機關加強宣導。
  三、本案擬函轉所屬機關配合辦理宣導事宜。

擬辦：如擬，陳請核示。

                              敬陳  ○長

承辦單位：○○科    承辦人：○○○    分機：○○○○
```

**函稿**：
```
○○○○○○○ 函（稿）

受文者：所屬各機關

主旨：轉知內政部函請加強防範一氧化碳中毒宣導一案，請
      配合辦理，請查照。

說明：
  一、依據內政部○○○年○月○日內授消字第○○○○號
      函辦理。
  二、請貴機關利用各種集會場合加強宣導。

正本：所屬各機關
副本：本府消防局
```

### 案例5：回復申請同意案

**情境**：某民間團體申請借用會議室，經審核符合規定。

**簽辦方式選擇理由**：
- 依法准駁案件
- 須說明審核情形
- 同時陳核簽與函

**簽**：
```
                簽稿併陳
                    簽

主旨：○○協會申請借用本府○樓會議室一案，經審核符合
      規定，擬同意借用，簽請核示。

說明：
  一、依據○○協會○○○年○月○日申請書辦理。
  二、該會擬於○○○年○月○日（星期○）上午○時至○
      時借用本府○樓會議室，召開會員大會。
  三、經查該會為依法設立之人民團體，申請程序符合「○
      ○○○借用管理要點」規定。

擬辦：
  一、擬同意借用。
  二、如擬，陳請核示。

                              敬陳  ○長

承辦單位：○○科    承辦人：○○○    分機：○○○○
```

**函稿**：
```
○○○○○○○ 函（稿）

受文者：○○協會

主旨：貴會申請借用本府○樓會議室一案，同意照辦，請查
      照。

說明：
  一、依據貴會○○○年○月○日申請書辦理。
  二、借用時間：○○○年○月○日（星期○）上午○時至
      ○時。
  三、請依「○○○○借用管理要點」相關規定使用。

正本：○○協會
副本：本府總務科
```

## 第五類：駁回及特殊案例

### 案例14：不同意申請案（簽稿併陳）

**情境**：民眾申請借用會議室，經審核不符規定。

**簽辦方式選擇理由**：
- 依法准駁案件
- 案情特殊須說明駁回理由
- 需說明不符規定之處

**簽**：
```
                簽稿併陳
                    簽

主旨：○○○君申請借用本府會議室一案，經審核與規定不符，
      擬不同意借用，簽請核示。

說明：
  一、依據○○○君○○○年○月○日申請書辦理。
  二、該君擬於○○○年○月○日借用本府○樓會議室。
  三、經查本案不符「○○○借用管理要點」第○條規定，
      理由如下：
    （一）申請人非屬依法設立之人民團體。
    （二）借用目的為商業活動，非屬公益性質。
  四、爰擬不同意借用。

擬辦：
  一、擬不同意借用。
  二、如擬，函復申請人。
  三、陳請核示。

                              敬陳  ○長

承辦單位：○○科    承辦人：○○○    分機：○○○○
```

**函稿**：
```
○○○○○○○ 函（稿）

受文者：○○○君

主旨：臺端申請借用本府會議室一案，與規定不符，礙難同意，
      請查照。

說明：
  一、依據臺端○○○年○月○日申請書辦理。
  二、本案不符「○○○借用管理要點」第○條規定，爰未便
      照准。

正本：○○○君
副本：
```
