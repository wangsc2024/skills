# 簽辦公文專家

本技能協助撰寫符合台灣政府機關規範的公文，依據公務人員保障暨培訓委員會《公文撰作解析》實作篇標準。

---

## 第一部分：公文基本概念

### 公文定義
依《公文程式條例》第1條，公文是指處理公務之文書。

### 公文類別
| 類別 | 用途 |
|------|------|
| **令** | 公布法律、任免獎懲官員、軍事機關發布命令 |
| **呈** | 對總統有所呈請或報告 |
| **咨** | 總統與立法院、監察院往復 |
| **函** | 各機關間公文往復，或人民與機關間申請與答復 |
| **公告** | 對公眾有所宣布 |
| **其他公文** | 書函、簽、開會通知單等 |

### 公文撰寫四大原則：簡、淺、明、確
1. **簡**：文字簡練，不堆砌詞藻
2. **淺**：用語淺顯，避免艱深術語
3. **明**：意思明白，條理分明
4. **確**：內容正確，語意肯定

## 第二部分：簽的撰擬

### 簽的定義
簽為內部溝通時使用的文書，屬上行文。承辦人員基於職責對職掌事務提出處理意見，以供上級瞭解案情並作為抉擇依據。

### 簽的格式

#### 一段式（案情簡單）
```
                    簽

主旨：○○○○○○○○○○，請核示。

                              敬陳  ○長
承辦單位：         承辦人：         分機：
```

#### 二段式（案情稍複雜）
```
                    簽

主旨：○○○○○○○○○○，請核示。

說明：
  一、依據○○○○。
  二、○○○○○○。

                              敬陳  ○長
承辦單位：         承辦人：         分機：
```

#### 三段式（案情複雜，標準格式）
```
                    簽

主旨：○○○○○○○○○○，請核示。

說明：
  一、依據○○○○○○。
  二、○○○○○○○○。
  三、○○○○○○○○。

擬辦：
  一、○○○○○○○○。
  二、○○○○○○○○。

                              敬陳  ○長
承辦單位：         承辦人：         分機：
```

### 簽各段撰寫要領

#### 主旨段
- **公式**：起首語 + 本案主要意旨 + 期望語
- **字數**：60字以內（最難寫，需摘要重點與洗鍊文字）
- **不分項**，一段完成
- **一文一事**：一件公文處理一件事
- **期望語選擇**：
  - 上行文：請鑒核、請核示（不可用「請核復」）
  - 平行文：請查照、請惠復、請卓參
  - 下行文：請照辦、希查照
- **注意**：期望語僅列於主旨段，不應於說明段、辦法段重複

#### 說明段（引據＋申述＋歸結）
- **引據**：依據法令、來文、計畫、會議決議
- **申述**：來龍去脈、人事時地物、分析利弊
- **歸結**：發文機關見解看法
- 可分項條列

#### 擬辦段
- 提出**具體處理意見**或解決方案
- 供長官核定或選擇
- 可分項條列

## 第三部分：簽辦方式判斷

### ⚠️ 判斷核心原則

**首要判斷：本案是否需要對外行文？**

| 需要對外行文 | 不需要對外行文 |
|-------------|---------------|
| 先簽後稿、簽稿併陳、以稿代簽 | **只簽不稿** |

> **常見錯誤**：內部經費動支、人事簽報、業務報告等案件，僅需簽奉核可即可執行，不需要對外發函，應使用「只簽不稿」，切勿誤用「簽稿併陳」。

### 簽辦方式判斷流程圖

```
案件性質判斷
    │
    ╔═══════════════════════════════════════════════════════════╗
    ║ 【第一關：是否需要對外行文？】                              ║
    ║                                                           ║
    ║  需要對外行文的情況：                                      ║
    ║  • 須函復來文機關或申請人                                  ║
    ║  • 須轉知所屬機關或單位                                    ║
    ║  • 須向上級機關報核                                        ║
    ║  • 須對外公告周知                                          ║
    ║                                                           ║
    ║  不需要對外行文的情況：                                    ║
    ║  • 內部經費動支案（奉核可即執行）                          ║
    ║  • 內部業務報告案                                          ║
    ║  • 內部人事簽報案                                          ║
    ║  • 內部作業程序簽核案                                      ║
    ╚═══════════════════════════════════════════════════════════╝
    │
    ├─ 不需要對外行文？ ─────────────────→ 【只簽不稿】
    │         │
    │         需要對外行文
    │         ↓
    ├─ 政策性/重大興革案件？ ─→ 是 ─→ 【先簽後稿】
    │         │
    │         否
    │         ↓
    ├─ 重要人事案件？ ─→ 是 ─→ 【先簽後稿】
    │         │
    │         否
    │         ↓
    ├─ 須說明或析述以往處理情形？ ─→ 是 ─→ 【簽稿併陳】
    │         │
    │         否
    │         ↓
    ├─ 依法准駁但案情特殊？ ─→ 是 ─→ 【簽稿併陳】
    │         │
    │         否
    │         ↓
    └─ 案情簡單或例行層轉？ ─→ 是 ─→ 【以稿代簽】
```

### 一、先簽後稿

**適用情況**：
- 制定、訂定、修正、廢止法令案件
- 有關政策性或重大興革案件
- 牽涉較廣，會商未獲結論案件
- 擬提決策會議討論案件
- 重要人事案件
- 涉及首長行程之案件

**文件置放順序**（由上至下）：
```
公文稿（上）
   ↓
簽（中）- 標註「先簽後稿」
   ↓
來文（下）
```

**案例類型**：
1. 向上層機關申請經費補助案
2. 邀請首長主持會議案
3. 人員商調案

---

### 二、簽稿併陳

> ⚠️ **重要提醒**：「簽稿併陳」顧名思義，**必須同時具備「簽」和「公文稿」兩份文件**。
> 若只寫簽而無公文稿，則為「只簽不稿」；若只寫公文稿而無簽，則為「以稿代簽」。

**適用情況**（須同時滿足：需對外行文 + 案情須說明）：
- 文稿內容須另作說明或析述以往處理情形
- 依法准駁但案情特殊須說明
- 須限時辦理不及先行請示

**文件置放順序**（由上至下）：
```
┌─────────────────────────────────┐
│  簽（上）- 標註「簽稿併陳」      │ ← 說明案情與擬辦意見
├─────────────────────────────────┤
│  公文稿（中）                    │ ← 對外行文的正式公文
├─────────────────────────────────┤
│  來文（下）                      │ ← 原始來文
└─────────────────────────────────┘
```

**案例類型**（皆需對外發函）：
1. 轉知宣導案（需函轉所屬）
2. 回復申請同意案（需函復申請人）
3. 回復申請並規劃後續案（需函復申請人）
4. 提報業務成果案（需函報上級機關）

**與「只簽不稿」的區別**：

| 比較項目 | 簽稿併陳 | 只簽不稿 |
|----------|----------|----------|
| **對外行文** | 需要 | 不需要 |
| **文件組成** | 簽 + 公文稿 | 僅簽 |
| **適用案例** | 函復申請人、轉知所屬 | 內部經費動支、業務報告 |

---

### 三、以稿代簽

**適用情況**：
- 案情簡單，毋須另作說明
- 例行層轉之案件
- 轉知法令規章

**作法**：
- 不另上簽，直接撰擬公文稿
- 於**公文稿面**加註「以稿代簽」
- **來文空白處**註明「以稿代簽」並蓋職名章

**文件置放順序**（由上至下）：
```
公文稿（上）- 標註「以稿代簽」
   ↓
來文（下）- 註明「以稿代簽」、蓋職名章
```

**案例類型**：
1. 移轉他機關辦理案
2. 函轉來文案
3. 轉知法令案
4. 例行函復案

---

### 四、只簽不稿

> ✅ **判斷要點**：本案奉核可後，是否可以直接執行而無需對外發函？
> 若「是」，則使用「只簽不稿」。

**適用情況**：
- 僅需簽報，無需對外行文
- 業務報告案
- 內部簽核案
- **內部經費動支案**（奉核可即執行）
- 內部人事簽報案
- 內部作業程序簽核案

**文件置放順序**（由上至下）：
```
┌─────────────────────────────────┐
│  簽                              │ ← 僅有簽，無公文稿
├─────────────────────────────────┤
│  相關附件（如有）                │ ← 佐證資料
└─────────────────────────────────┘
```

**案例類型**：
1. 簽陳業務報告案
2. 簽報無需行文案
3. **內部經費動支案**
4. 內部採購簽核案
5. 人員差假簽核案
6. 內部作業流程簽核案

### 只簽不稿範例：內部經費動支案

**情境**：某處室申請辦公室整修經費 50 萬元。

```
                    簽

主旨：本處辦公室設備老舊亟需整修，擬動支整修經費新臺幣
      50萬元，簽請核示。

說明：
  一、本處辦公空間自民國95年啟用迄今已逾18年，相關設備
      設施已嚴重老化，影響同仁辦公效率與服務品質。
  二、經本處評估，急需整修項目包括：
    （一）冷氣空調設備更新。
    （二）辦公桌椅汰換。
    （三）照明設備改善。
  三、上開整修預估經費新臺幣50萬元整，經費來源擬由本年
      度預算「房屋建築及設備費」科目支應。

擬辦：
  一、擬同意動支上開經費辦理整修。
  二、奉核可後，由本處逕行辦理後續採購作業。
  三、陳請核示。

                              敬陳  縣長

承辦單位：○○處    承辦人：○○○    分機：○○○○
```

**為何使用「只簽不稿」**：
- ✅ 本案為內部經費動支
- ✅ 奉核可後可直接辦理採購，無需對外發函
- ✅ 不需要函復任何機關或申請人

## 第五部分：書函的撰擬

### 書函定義
書函用於公務未決階段需要磋商、徵詢意見或通報時使用，代替過去之便函、備忘錄、簡便行文表。

### 書函與函的差異

| 比較項目 | 函 | 書函 |
|----------|-----|------|
| **正式性** | 正式公文 | 性質較不正式 |
| **使用時機** | 各機關公文往復、人民與機關申請答復 | 公務未決階段磋商、徵詢、通報 |
| **適用情況** | 正式決定事項 | 答復簡單案情、寄送普通文件、一般聯繫查詢 |
| **用印** | 蓋用印信 | 得以簡便方式處理 |

### 書函適用情況
1. 公務未決階段需要磋商、徵詢意見
2. 答復簡單案情
3. 寄送普通文件、書刊
4. 一般聯繫、查詢事項
5. 機關對民間團體行文

### 書函格式
```
○○○○○○○ 書函
                                    地址：○○○○○○○○
                                    聯絡方式：（承辦人、電話、email）

受文者：○○○○○○

發文日期：中華民國○○○年○○月○○日
發文字號：○○字第○○○○○○○○○○號
速別：普通件
附件：（如有）

主旨：[行文目的，60字內]

說明：
  一、[引據]
  二、[申述]

                                        ○長 ○○○
```

### 書函範例

**情境**：詢問某單位意見，尚在研議階段

```
○○○○○○○ 書函

受文者：○○○單位

發文日期：中華民國○○○年○月○日
發文字號：○字第○○○○○○○○○號

說明：
  一、本府擬辦理○○○事項，目前尚在研議階段。
  二、為周延考量，擬請貴單位就○○○部分提供意見，
      俾供參酌。

                                        ○長 ○○○
```

## 第七部分：公告的撰擬

### 公告定義
公告為對公眾有所宣布時使用，主要用於法規發布、人事任命、機關公告等。

### 公告格式
```
○○○○○○○ 公告
                                    發文日期：中華民國○○○年○○月○○日
                                    發文字號：○○字第○○○○○○○○○○號

主旨：[公告事項，60字內]

依據：（法令依據或授權，可省略）

公告事項：
  一、○○○○○○○○。
  二、○○○○○○○○。
  三、○○○○○○○○。

                                        ○長 ○○○
```

### 公告用印規定
| 張貼位置 | 用印規定 |
|----------|----------|
| 機關公布欄 | 須蓋用機關印信及署首長職銜、姓名 |
| 政府公報 | 免蓋用印信 |
| 電子公布欄 | 免蓋用印信 |

### 公告範例

**情境**：公告法規修正

```
○○○縣政府 公告
                                    發文日期：中華民國○○○年○月○日
                                    發文字號：屏府○字第○○○○○○○○號

主旨：公告修正「○○○縣○○○自治條例」部分條文。

依據：地方制度法第25條、第26條規定。

公告事項：
  一、修正「○○○縣○○○自治條例」第5條、第8條條文。
  二、修正條文自公告日施行。
  三、附修正條文對照表。

                                        縣長 ○○○
```

## 第九部分：移文單

### 移文單定義
移文單用於來文非本機關職權所能處理，須將原案移請權責機關處理時使用。

### 使用時機
1. 不屬於本機關主管業務或職權範圍之來文，逕以移文單移送主管機關
2. 承辦單位收受之文件非屬本單位承辦者，敘明理由移送其他單位

### 移文單格式
```
○○○○○○○ 移文單
                                    機關地址：○○○○○○○○
                                    聯絡電話：○○○○○○○○

受文者：○○○○○○

發文日期：中華民國○○○年○○月○○日
發文字號：○○字第○○○○○○○○○○號

主旨：○○○（來文機關）○○○年○月○日○字第○○○號函，
      為○○○事，案屬貴管，移請卓辦。

附件：原函影本○份

                                        （蓋機關條戳）
```

### 注意事項
1. **只需主旨一段**：無須說明及辦法段
2. **蓋用條戳**：不用首長職銜簽字章
3. **受移單位不得再移**：有意見應簽請首長協調判定
4. **附來文影本**：便於受移機關辦理

### 移文單範例

**情境**：民眾陳情事項非本機關權責

```

發文字號：屏府秘字第○○○○○○○○號
```

## 第十部分：分層負責與核決

### 分層負責概念
各機關為實施分層負責、逐級授權，依中央行政機關組織基準法第8條規定，得訂定分層負責明細表。

### 行文名義規定
| 決定層級 | 行文名義 | 說明 |
|----------|----------|------|
| 機關首長核定 | 機關名義 | 以機關全銜行文 |
| 第2層主管核定 | 機關名義 | 性質宜用機關名義者 |
| 第3層主管核定 | 機關或單位名義 | 視案件性質決定 |
| 單位主管逕行決定 | 單位名義 | 以該單位名義行文 |

### 用印規定
1. 任何文件非經機關首長或依分層負責規定授權主管判發者，不得蓋用印信
2. 監印人員發現原稿未經判行或有錯誤，應退送補判或更正後再蓋印
3. 電子公文依電子簽章法規定辦理

### 陳核原則
1. 按規定權責陳核，不越級陳核
2. 緊急案件得逕陳首長，但應副知中間層級
3. 越級行文須副本知會直屬上下級機關

### 會辦規定
1. **先簽後稿已會核者**：如稿內與會核時無出入，不再送會
2. **會簽視為同意**：各單位如未提出意見，經會簽即認為同意，共同負責
3. **不同意見處理**：由主辦單位綜合修改後，再送決定
4. **主辦單位責任**：送會前應先敘明本單位具體意見

### 會辦流程
```
承辦人 → 承辦人直屬2級主管 → 承辦人直屬1級主管
    → 各會辦單位 → 陳核決行主管
```

## 第十部分：公文時效管制

### 一般公文處理時限
| 速別 | 處理時限 | 說明 |
|------|----------|------|
| **最速件** | 1日 | 以工作天計算，扣除假日 |
| **速件** | 3日 | 以工作天計算，扣除假日 |
| **普通件** | 6日 | 以工作天計算，扣除假日 |

### 限期公文處理
- 依來文所訂期限為準
- 以日曆天計算（包含假日）
- 截止日為假日者，遞延至下一工作日

### 特殊案件時限
| 案件類型 | 計算方式 |
|----------|----------|
| 一般公文 | 扣除假日（工作天） |
| 限期公文 | 含假日（日曆天） |
| 人民陳情案件 | 扣除假日（工作天） |
| 人民申請案件 | 含假日（日曆天） |
| 訴願案件 | 含假日（日曆天） |

### 展期規定
1. 預計無法於時限內辦結，應於期限前簽請展期
2. 人民申請案件展期以一次為限
3. 展期日數不得超過原處理時限
4. 應於原期限屆滿前通知申請人

### 逾期公文處理
- 逾期公文列入管考
- 影響機關考核績效
- 承辦人員應負相關責任

## 第十三部分：常見錯誤與修正

### ⚠️ 零、簽辦方式選擇錯誤（最常見！）

> 簽辦方式選錯是公文撰擬最常見的錯誤之一，務必先確認是否需要對外行文。

#### 錯誤0-1：內部經費動支案誤用「簽稿併陳」
❌ **錯誤做法**：
```
                簽稿併陳
                    簽

主旨：本處辦公室整修，擬申請經費50萬元，簽請核示。
...
```
**問題**：「簽稿併陳」必須同時有簽和公文稿，但內部經費動支案奉核可即可執行，無需對外發函，根本不需要公文稿。

✅ **正確做法**：使用「只簽不稿」
```
                    簽

主旨：本處辦公室整修，擬動支經費50萬元，簽請核示。
...
```

#### 錯誤0-2：標註「簽稿併陳」卻只寫簽
❌ **錯誤做法**：標註「簽稿併陳」但文件中只有簽，沒有附上公文稿
**問題**：「簽稿併陳」顧名思義必須有「簽」+「公文稿」兩份文件

✅ **正確做法**：
- 若需對外行文：補上公文稿，維持「簽稿併陳」
- 若不需對外行文：改為「只簽不稿」，移除標註

#### 錯誤0-3：需對外行文卻用「只簽不稿」
❌ **錯誤做法**：上級機關來函要求回報執行成果，卻只簽奉核不發函
**問題**：上級機關來函需要函復，不能只簽不稿

✅ **正確做法**：使用「簽稿併陳」，簽陳案情說明並附上函復上級機關的公文稿

#### 簽辦方式選擇速查表

| 案件類型 | 需對外行文？ | 正確簽辦方式 |
|----------|-------------|-------------|
| 內部經費動支 | 否 | **只簽不稿** |
| 內部業務報告 | 否 | **只簽不稿** |
| 人員差假簽核 | 否 | **只簽不稿** |
| 函復上級來函 | 是 | 先簽後稿／簽稿併陳 |
| 轉知所屬單位 | 是 | 簽稿併陳／以稿代簽 |
| 函復申請人 | 是 | 簽稿併陳／以稿代簽 |
| 向上級申請經費補助 | 是 | **先簽後稿** |

### 一、格式錯誤

#### 錯誤1：主旨分項
❌ 主旨：一、○○○。二、○○○。請核示。
✅ 主旨：○○○○○○○○○○，請核示。

#### 錯誤2：主旨超過60字
❌ 主旨過長，包含過多細節
✅ 將細節移至說明段

#### 錯誤3：段名後空格
❌ 主旨： ○○○（有空格）
✅ 主旨：○○○（緊接冒號）

#### 錯誤4：橫式用「如左」
❌ 說明如左：
✅ 說明如下：

### 二、邏輯錯誤

#### 錯誤5：說明段無邏輯順序
❌ 說明內容跳躍，無引據申述歸結
✅ 依序：引據→申述→歸結

#### 錯誤6：擬辦無具體建議
❌ 擬辦：請核示。
✅ 擬辦：一、擬同意辦理。二、陳請核示。

#### 錯誤7：說明與辦法混淆
❌ 說明段包含具體要求或辦法
✅ 說明段僅敘述案由，具體作法移至辦法段

### 三、用語錯誤

#### 錯誤8：期望語與對象不符
❌ 對上級機關用「請查照」
✅ 對上級機關用「請鑒核」或「請核示」

#### 錯誤9：稱謂語不一致
❌ 前稱「張先生」，後改「張君」
✅ 同一公文內稱謂應前後一致

#### 錯誤10：雙重否定語氣
❌ 似無不可、尚無不合、似可照准
✅ 應予照准、符合規定、准予照辦

#### 錯誤11：引敘語與對象不符
❌ 對上級機關用「據」
✅ 對上級機關用「奉」

### 四、引用錯誤

#### 錯誤12：自行重組法令原文
❌ 依據政府採購法規定，機關辦理採購...（自行改寫）
✅ 依據「政府採購法」第○條規定：「...」（照錄原文並加引號）

#### 錯誤13：引述大意不加「略以」
❌ 依據○○部函，各機關應...
✅ 依據○○部函略以，各機關應...

### 五、副本處理錯誤

#### 錯誤14：副本未註明含附件
❌ 副本：本府各機關（副本需附件但未註明）
✅ 副本：本府各機關（含附件）

#### 錯誤15：副本配合事項未說明
❌ 副本抄送但未說明需配合事項
✅ 說明段加註「副本抄送○○，請配合辦理」
