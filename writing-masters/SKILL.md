# 西方文學巨匠寫作技巧 Skill

匯集海明威、史蒂芬·金、乔治·奧威爾、托爾斯泰、乔斯夫·康貝爾等文學大師的寫作精華，提供實用的創作技巧與心法。

## 適用情境

當你需要：
- 提升文字的力量與精準度
- 學習「少即是多」的極簡寫作風格
- 掌握「展示而非告知」(Show, don't tell) 技巧
- 運用冰山理論創造深度
- 改善對話和敘事
- 建立有效的寫作習慣
- 進行有效的修訂與編輯

## 核心大師與技法

### 1. 海明威 (Ernest Hemingway) - 冰山理論

**核心理念：** 故事的深層意義藏在表面之下，只展現冰山一角，其餘留給讀者推斷。

> "如果一個作家對他所寫的東西有足夠的了解，他可以省略他所知道的東西，讀者仍會對這些有強烈的感覺，如同作家已經陳述了它們一樣。"

**關鍵技巧：**

```
冰山理論實踐：
┌─────────────────────────────┐
│     水面上 (1/8)           │ ← 文字所呈現的
│   對話、動作、細節         │
├─────────────────────────────┤
│                             │
│     水面下 (7/8)           │ ← 隱含的意義
│   情感、背景、動機         │
│   未說出口的真相           │
│   讀者自行推斷的內容       │
│                             │
└─────────────────────────────┘
```

**實踐要點：**

1. **省略是刻意的，不是疏忽**
   - 你必須知道被省略的部分
   - 省略是因為信任讀者的智慧
   - 空白處應該「充滿意義」

2. **用具體細節暗示情感**
   ```
   ❌ 他很悲傷。
   ✓ 他把杯子放下，看著窗外的雨，沒有說話。
   ```

3. **對話承載潛台詞**
   ```
   ❌ "我愛你，但我必須離開，這讓我很痛苦。"
   ✓ "火車幾點？" "六點。" "那我送你。" "不用了。"
   ```

### 2. 海明威的文體風格

**極簡主義寫作原則：**

| 原則 | 說明 |
|------|------|
| **簡潔詞彙** | 使用具體、常見、盎格魯撒克遜詞彙 |
| **短句優先** | 力量來自短句和精確細節 |
| **少用形容詞** | 讓名詞和動詞說話 |
| **避免抽象** | 用具體意象取代抽象概念 |
| **節制副詞** | 副詞是懶惰的標誌 |

**海明威的修訂哲學：**
> "我寫一頁傑作，配九十一頁垃圾。我試著把垃圾丟進垃圾桶。"

- 《乔别離武器》結局修改了 39 次
- 完成初稿只是開始
- 真正的寫作在修訂中完成

### 3. 史蒂芬·金 (Stephen King) - 寫作實踐

**《論寫作》核心教導：**

**基本功：**
```
每天閱讀 + 每天寫作 = 進步的唯一公式
```

**關鍵建議：**

1. **大量閱讀**
   - 金每年讀 70-80 本書
   - 閱讀教你風格、對話、描寫的長度
   - 也教你什麼不該做

2. **使用你的聲音**
   - 用腦中浮現的第一個詞
   - 不要強迫使用超出詞彙表的詞
   - 簡單的詞彙表沒什麼可恥的

3. **避免被動語態和副詞**
   ```
   ❌ "他輕輕地說道。" (副詞)
   ✓ "他說。" 或用動作展示輕柔

   ❌ "門被他推開了。" (被動)
   ✓ "他推開門。" (主動)
   ```

4. **關上門寫初稿**
   - 初稿不要接受他人意見
   - 完成後再開門接受回饋
   - 第一讀者應該是你信任的人

5. **故事是化石**
   > "故事是遺物，是未被發現的既存世界的一部分。作家的工作是用工具箱裡的工具，盡可能完整地把每個故事挖掘出來。"

**金的寫作公式：**
```
第二稿 = 第一稿 - 10%
```

### 4. 乔治·奧威爾 (George Orwell) - 清晰寫作六原則

**《政治與英語語言》六條規則：**

1. **永不使用已成陳腔濫調的隱喻或明喻**
2. **能用短詞時絕不用長詞**
3. **能刪的詞一定要刪**
4. **能用主動語態時絕不用被動**
5. **能用日常詞時絕不用外來詞、科學詞或行話**
6. **寧可打破以上規則，也不說野蠻的話**

**奧威爾的終極檢驗：**
> "一個細心的作家在每寫一句話時都應該問自己至少四個問題：
> 1. 我想說什麼？
> 2. 用什麼詞能表達它？
> 3. 什麼意象或習語能使它更清晰？
> 4. 這個意象夠新鮮嗎？"

### 5. 托爾斯泰 (Leo Tolstoy) - 角色深度

**《安娜·卡列尼娜》的創作啟示：**

- 歷時四年，多次修訂
- 從「墮落女人」的膚淺處理，演變為對安娜的細膩同情
- 移除大部分背景細節，代之以暗示和建議
- 將她從可憐可悲提升為值得同理的女主角

**技巧：**
- 讓角色配得上讀者的同理心
- 減少說明，增加暗示
- 讓讀者自己發現角色的複雜性

### 6. 費茲傑羅 (F. Scott Fitzgerald) - 精雕細琢

**《乔了不起的蓋茨比》寫作風格：**
- 每個句子都經過精心設計
- 象徵性元素編織其中
- 散文具有詩的韻律

> 海明威評價：費茲傑羅是「真正的作家」

## 展示而非告知 (Show, Don't Tell)

**核心原則：**
```
告知 = 作者直接陳述
展示 = 讓讀者通過細節推斷
```

**實踐對照：**

| 告知 (Tell) | 展示 (Show) |
|-------------|-------------|
| 她很害怕 | 她的手在發抖，杯子裡的水晃動著 |
| 他很憤怒 | 他把椅子踢開，文件散落一地 |
| 天氣很熱 | 柏油路面閃著熱浪，狗趴在陰影裡喘氣 |
| 他們相愛 | 他們的手指在桌下悄悄觸碰 |

**海明威的示範：**
> "用事實般的聲音和精簡的描述，場景反而更令人不寒而慄。"

## 大師的寫作習慣

### 海明威
- 每天早晨寫作
- 站著寫
- 每天統計字數
- 在知道下一步要寫什麼時停筆

### 史蒂芬·金
- 每天寫 2000 字
- 早上寫作
- 初稿應在三個月內完成
- 完成後放置六週再修改

### 共同建議
- **設定固定時間**：寫作是工作，不是等待靈感
- **每日練習**：肌肉需要鍛鍊，寫作亦然
- **完成比完美重要**：先寫完，再修改
- **為自己寫初稿**：不要想著讀者會怎麼說

## 修訂技巧

### 層次化修訂法

```
第一輪：結構
├── 故事邏輯是否通順？
├── 場景順序是否合理？
└── 是否有多餘的章節？

第二輪：場景
├── 每個場景是否推動情節？
├── 開場是否吸引人？
└── 結尾是否有力？

第三輪：段落和句子
├── 是否有過長的段落？
├── 句子是否多樣？
└── 是否有重複？

第四輪：用詞
├── 是否有更好的動詞？
├── 形容詞是否必要？
└── 是否有陳腔濫調？
```

### 海明威的修訂清單

- [ ] 刪除所有不必要的副詞
- [ ] 將被動語態改為主動
- [ ] 用具體名詞取代抽象名詞
- [ ] 縮短過長的句子
- [ ] 移除解釋性文字，留下動作
- [ ] 檢查對話是否自然

### 金的 10% 法則

```
初稿字數 × 0.9 = 目標字數

如果初稿 10,000 字 → 第二稿 9,000 字
刪除 1,000 字的方法：
- 重複的資訊
- 過度的描述
- 不推動情節的對話
- 解釋讀者已理解的內容
```

## 對話寫作

### 海明威風格對話

**特點：**
- 簡短、精準
- 省略說話標籤（他說、她說）
- 潛台詞比明說更重要

```
"你要走了。"
"是的。"
"什麼時候？"
"明天早上。"
她沒有說話。他們聽著外面的雨聲。
"要我送你嗎？"
"不用。"
```

### 金的對話原則

1. **每個角色有獨特的聲音**
2. **對話要推動情節**
3. **少用副詞標籤**
   ```
   ❌ "我恨你，"她憤怒地說。
   ✓ "我恨你。" 她把門摔上。
   ```

## 快速參考

### 開始寫作前
- 讀一段喜歡的作家的文字
- 忘記批評，先寫完
- 想著一個具體的讀者

### 寫作中
- 使用具體細節
- 信任讀者的智慧
- 寫你知道的（或研究透徹的）

### 完成後
- 放置一段時間再修改
- 大聲朗讀
- 刪除 10% 的內容
- 每個詞都要證明它存在的必要
