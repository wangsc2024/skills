# 熱力學熵深度參考

## 歷史發展

### 1. Clausius 與熵的定義 (1865)

Rudolf Clausius 定義熵的變化量：

```
dS = δQ_rev / T

其中：
- dS = 熵的微分變化
- δQ_rev = 可逆過程中的熱量傳遞
- T = 絕對溫度
```

**Clausius 不等式：**
```
∮ δQ/T ≤ 0

等號成立於可逆循環
```

### 2. Boltzmann 統計力學詮釋 (1877)

```
S = k_B × ln W

這個公式刻在 Boltzmann 的墓碑上
```

**W 的計算：**
對於 N 個粒子分佈在 n 個能級上：
```
W = N! / (n₁! × n₂! × ... × nₖ!)
```

### 3. Gibbs 熵公式

```
S = -k_B × Σᵢ pᵢ × ln pᵢ

這是 Shannon 熵的物理版本
```

## 熱力學第二定律的表述

### Clausius 表述
> 熱量不能自發地從低溫物體傳向高溫物體

### Kelvin-Planck 表述
> 不可能從單一熱源吸熱並將其完全轉化為功而不產生其他影響

### 熵增原理表述
> 孤立系統的熵永不減少

### 數學表述
```
對於孤立系統：dS ≥ 0
對於非孤立系統：dS_系統 + dS_環境 ≥ 0
```

## 熵與自由能

### Helmholtz 自由能
```
F = U - TS

其中 U 是內能
```

### Gibbs 自由能
```
G = H - TS = U + PV - TS

自發過程：dG < 0（恆溫恆壓）
```

### 熵與平衡
```
在平衡態：dS_total = 0
```

## 生命與熵

### Schrödinger 的觀點 (1944)

在《生命是什麼》中提出：

> 生命以負熵為食（"What an organism feeds upon is negative entropy"）

**生命維持低熵的代價：**
```
ΔS_生命 < 0（局部）
ΔS_環境 > |ΔS_生命|（更大的增加）
ΔS_總體 > 0（符合第二定律）
```

### 耗散結構 (Prigogine)

遠離平衡態的開放系統可以形成有序結構：
- 對流胞（Bénard cells）
- 化學振盪（如 BZ 反應）
- 生命本身

## Maxwell 妖與 Landauer 原理

### Maxwell 妖（1867）

假想的智慧生物，能區分快慢分子：

```
   熱浴                    熱浴
[高溫] ◄─── [妖] ───► [低溫]
         只讓快分子右移
         只讓慢分子左移
```

看似違反第二定律，因為不做功就創造了溫度差。

### Szilard 引擎（1929）

將 Maxwell 妖簡化為可計算的模型，證明獲取信息需要能量。

### Landauer 原理（1961）

> 擦除 1 bit 信息至少需要 kT ln(2) 能量

```
E_min = k_B × T × ln(2) ≈ 2.87 × 10^-21 J（室溫）
```

這解決了 Maxwell 妖悖論：妖需要記憶（存儲信息），最終必須擦除記憶，這就會增加熵。

## 熵的測量

### 熱量計方法
```
ΔS = ∫(C_p / T) dT（恆壓）
ΔS = ∫(C_v / T) dT（恆容）
```

### 第三定律（Nernst 定理）
```
lim(T→0) S = S₀ = 0（對於完美晶體）
```

這允許計算絕對熵。

### 標準熵值範例

| 物質 | S° (J/mol·K) |
|------|--------------|
| C(石墨) | 5.7 |
| C(鑽石) | 2.4 |
| H₂O(液) | 69.9 |
| H₂O(氣) | 188.8 |
| O₂(氣) | 205.1 |
| CO₂(氣) | 213.7 |

## 實際應用

### 熱機效率

**卡諾效率：**
```
η_Carnot = 1 - T_cold / T_hot
```

這是所有熱機效率的理論上限。

### 製冷係數

```
COP_refrigerator = T_cold / (T_hot - T_cold)
COP_heat_pump = T_hot / (T_hot - T_cold)
```

### 化學反應可行性

```
ΔG = ΔH - TΔS

ΔG < 0 → 自發
ΔG = 0 → 平衡
ΔG > 0 → 非自發
```

| ΔH | ΔS | ΔG | 反應特性 |
|----|----|----|----------|
| - | + | 總是 < 0 | 總是自發 |
| + | - | 總是 > 0 | 總不自發 |
| - | - | 取決於 T | 低溫自發 |
| + | + | 取決於 T | 高溫自發 |
