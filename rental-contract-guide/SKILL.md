# 租賃契約專家 Skill

台灣不動產租賃法律體系、條款設計與專家系統建置指南。

## When to Use This Skill

- 撰寫或審閱租賃契約
- 檢查契約條款合規性（契約掃毒）
- 處理租賃糾紛（欠租、損壞、押金返還）
- 計算電費（2024 新制）
- 計算設備折舊賠償
- 生成存證信函範本
- 了解租賃法規與權益

---

## 核心功能模組

### 1. 合規性檢核（契約掃毒）

自動檢查契約是否符合：
- 《租賃住宅市場發展及管理條例》
- 《住宅租賃定型化契約應記載及不得記載事項》
- 2024 年電費新制

**檢核項目：**
- 是否違反「不得記載事項」（如限制報稅、禁止遷入戶籍、押金超過2個月）
- 是否包含所有「應記載事項」（如審閱期、提前終止規範）

### 2. 電費計算器（2024 新制）

```
輸入：用電度數、台電帳單金額
輸出：每度平均電價、合法收費上限
```

**規則：**
- 按「度」計費：不得超過「當期每度平均電價」
- 非按「度」計費：總額不得超過帳單總額
- 房客有權向台電查詢電費資訊

### 3. 押金檢核

- 上限：2 個月租金
- 超過部分無效

### 4. 折舊計算器（平均法）

```
輸入：物品名稱、原價、耐用年數、已使用年數
輸出：殘值、建議賠償金額
```

**常見設備耐用年數：**
| 設備 | 耐用年數 |
|------|----------|
| 冷氣 | 5-8 年 |
| 冰箱 | 8-10 年 |
| 洗衣機 | 6-8 年 |
| 沙發 | 5-7 年 |
| 床墊 | 7-10 年 |

## 智能條款工程

### 修繕責任劃分

| 類別 | 房東責任 | 房客責任 |
|------|----------|----------|
| 結構設備 | 屋頂、牆壁、管線 | - |
| 固定設備 | 冷氣、熱水器故障 | 濾網清潔 |
| 消耗品 | - | 燈泡、電池 |
| 人為損壞 | - | 全額負責 |

### 寵物條款設計

```markdown
## 寵物飼養規範
1. 寵物履歷：品種、體型、預防針證明
2. 清潔保證金：額外 0.5-1 個月
3. 損害責任：超出正常耗損由房客負責
4. 禁養類型：大型犬、攻擊性動物
```

### 生活公約

- 噪音管制：晚間 10 點後
- 抽菸規範：室內禁菸或指定區域
- 垃圾處理：依規定時間分類

## 重要法規摘要

### 應記載事項

- 契約審閱期 ≥ 3 日
- 押金 ≤ 2 個月租金
- 租期內不得調漲租金
- 提前終止需 1 個月前通知

### 不得記載事項

- 禁止報稅或遷入戶籍
- 拋棄審閱期間
- 排除瑕疵擔保責任
- 約定繳回契約書

## 2024-2025 法規變革

### 電費新制 (2024/7/15)

- 每度單價不得超過「當期平均電價」
- 房客有權向台電查詢電費資訊
- 違規罰鍰：3-30 萬元

### 租期保障草案 (2025)

- 房客至少 3 年租賃保障
- 續約漲幅限制（房租指數年增率）
- 強化租霸驅離機制

## 糾紛處理導航

### 存證信函範本

#### 催欠租

```
主旨：催繳租金通知

台端承租本人位於 [地址] 之房屋，依租賃契約約定，
每月租金新台幣 [金額] 元整，應於每月 [日期] 前繳納。

惟台端自 [年月] 起，已積欠 [月數] 個月租金，
合計新台幣 [金額] 元整。

請台端於收到本函後 [7/14] 日內繳清積欠租金，
逾期未繳，本人將依法終止租約並請求損害賠償。

此致
[承租人姓名]
```

#### 催修繕

```
主旨：請求修繕通知

台端出租予本人之房屋，位於 [地址]，
其 [設備名稱] 於 [日期] 發生故障，影響正常使用。

依民法第 429 條及租賃契約約定，
出租人負有修繕義務。

請台端於收到本函後 [14] 日內完成修繕，
逾期本人將自行僱工修繕並於租金中扣除。

此致
[出租人姓名]
```

#### 返還押金

```
主旨：請求返還押金

本人與台端之租賃契約已於 [日期] 終止，
並已依約完成點交手續。

依契約約定，台端應於點交後 [期限] 內返還押金
新台幣 [金額] 元整。

請台端於收到本函後 [14] 日內返還押金，
逾期本人將依法追訴。

此致
[出租人姓名]
```

## 公證與強制執行

### 何時需要公證

- 租期超過 5 年
- 希望逕受強制執行（免訴訟驅離）
- 金額較大的商業租賃

### 公證效力

- 租約期滿或終止時，可直接聲請強制執行
- 無需另行訴訟
- 費用：依公證法收費標準

---

## 注意事項

- 始終以最新版內政部定型化契約應記載及不得記載事項為準
- 涉及複雜法律訴訟時，應提醒使用者諮詢專業律師
- 電費計算以台電公告為準
