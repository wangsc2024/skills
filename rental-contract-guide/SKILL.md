# 租賃契約專家 Skill

台灣不動產租賃法律體系、條款設計與專家系統建置指南。

## When to Use This Skill

- 撰寫或審閱租賃契約
- 檢查契約條款合規性
- 處理租賃糾紛（欠租、損壞、押金返還）
- 計算電費（2024 新制）
- 了解租賃法規與權益

## 核心功能模組

### 1. 合規性檢核

自動檢查契約是否符合：
- 《租賃住宅市場發展及管理條例》
- 《住宅租賃定型化契約應記載及不得記載事項》
- 2024 年電費新制

### 2. 電費計算器

```
輸入：用電度數、台電帳單金額
輸出：每度平均電價、合法收費上限
```

### 3. 押金檢核

- 上限：2 個月租金
- 超過部分無效

### 4. 折舊計算器

```
輸入：物品名稱、原價、已使用年數
輸出：建議賠償金額（扣除折舊）
```

## 重要法規摘要

### 應記載事項

- 契約審閱期 ≥ 3 日
- 押金 ≤ 2 個月租金
- 租期內不得調漲租金
- 提前終止需 1 個月前通知

### 不得記載事項

- 禁止報稅或遷入戶籍
- 拋棄審閱期間
- 排除瑕疵擔保責任
- 約定繳回契約書

## 2024-2025 法規變革

### 電費新制 (2024/7/15)

- 每度單價不得超過「當期平均電價」
- 房客有權向台電查詢電費資訊
- 違規罰鍰：3-30 萬元

### 租期保障草案 (2025)

- 房客至少 3 年租賃保障
- 續約漲幅限制（房租指數年增率）
- 強化租霸驅離機制
