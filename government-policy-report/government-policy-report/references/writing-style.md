# 施政總報告撰寫風格指南

本文件提供施政總報告的語言風格、用詞規範及常見範例。

## 語言風格原則

### 1. 正式書面語
施政報告為正式公文書，應使用正規書面語言。

**正確用法**：
- 本府、本縣、本市
- 推動、辦理、執行
- 依據、基於、鑑於

**避免用法**：
- 我們縣、我們這邊
- 做了、弄了
- 因為所以、然後

### 2. 謙遜態度
首長報告應展現謙遜態度。

**正確用法**：
- 「承蒙議會支持」
- 「敬請各位議員指教」
- 「在各位議員的監督下」

**避免用法**：
- 「我們做得很好」
- 「這是我的功勞」
- 「議會應該配合」

### 3. 積極正面
聚焦成果與展望，避免消極或指責。

**正確用法**：
- 「持續改善中」
- 「未來將積極推動」
- 「面對挑戰，本府已規劃...」

**避免用法**：
- 「前任政府留下的問題」
- 「經費不足無法執行」
- 「中央不給補助」

```
推動、辦理、執行、落實、進行、實施、開展、啟動
```

### 表達完成達成
```
完成、達成、實現、完竣、順利執行、如期完工
```

### 表達提升改善
```

```

```
規劃、籌劃、研擬、評估、將持續、積極爭取
```

## 常見句型範例

### 開場句型

```
欣逢貴會第○屆第○次定期大會開議，○○謹代表縣府團隊，向大會提出施政總報告，敬請指教。
```

```
承蒙議會長期以來的支持與監督，本府過去一年在各項施政工作上，均有具體成效。
```

```
過去○個月，本府秉持「○○○」的施政理念，積極推動各項建設，茲將施政成果報告如下。
```

### 成果陳述句型

```
在○○方面，本府已完成○○○，受益民眾達○○萬人。
```

```
【○○建設】部分，累計投入經費○○億元，完成○○公里道路改善。
```

```
○○政策實施以來，成效顯著，○○○由○○提升至○○，成長○○%。
```

### 數據佐證句型

```
根據統計，本縣○○○較去年同期成長○○%，顯示○○政策已見成效。
```

```
截至○年○月底，已完成○○○，達成率○○%，超越預定目標。
```

```
經第三方調查，民眾滿意度達○○%，較前年提升○個百分點。
```

### 未來展望句型

```
展望未來，本府將持續推動○○○，預計於○年○月前達成○○目標。
```

```
針對○○議題，本府已研擬○○計畫，將於下一會期向議會專案報告。
```

```
為因應○○挑戰，本府將積極爭取中央補助，並籌措自籌款○○億元推動○○建設。
```

### 結語句型

```
縣政推動需要府會攜手合作，期盼各位議員繼續給予支持與指教。
```

```
以上施政總報告，敬請各位議員不吝指教。謝謝！
```

```
○○將帶領縣府團隊，以更積極的態度、更務實的作為，為○○縣民謀求最大福祉。
```

## 章節轉換用語

### 章節開始
- 首先，報告○○方面的施政成果...
- 在○○領域，本府推動成果如下...
- 接下來，報告○○部分...

### 章節結束
- 以上為○○方面的施政報告。
- ○○領域之施政成果簡報如上。
- 有關○○部分，謹報告至此。

### 轉折過渡
- 除上述成果外，本府亦積極推動...
- 另在○○方面...
- 此外，針對議員關心的○○議題...

## 常見錯誤用法

### 用字錯誤
| 錯誤 | 正確 |
|------|------|
| 施政計劃 | 施政計畫 |
| 會議記錄 | 會議紀錄 |
| 檢查 | 稽查（稅務/環保用語） |
| 做好 | 妥善處理 |

### 語病
| 錯誤 | 正確 |
|------|------|
| 大約有1,234人 | 約1,200人/計1,234人 |
| 已經完成達成 | 已完成/已達成 |
| 提高增加 | 提高/增加（擇一） |

### 態度不當
| 錯誤 | 正確 |
|------|------|
| 議會應該支持 | 敬請議會支持 |
| 這不是我們的問題 | 本府將積極協調處理 |
| 前任政府留下的 | （不提及，聚焦當前施政） |

## 敏感議題處理

### 預算不足
**建議寫法**：
```
在有限資源下，本府優先推動○○，並積極爭取中央補助...
```

### 工程延遲
**建議寫法**：
```
因應○○因素，本府已調整工期，預計○年○月完工，並強化進度管控...
```

### 民怨議題
**建議寫法**：
```
針對民眾關切的○○議題，本府已成立專案小組，研擬改善方案...
```

### 中央地方權責
**建議寫法**：
```
本案涉及中央權責，本府已函請○○部（會）協助，並持續追蹤辦理...
```
