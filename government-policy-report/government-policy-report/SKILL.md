# 台灣地方政府施政總報告撰寫指南

本技能協助撰寫符合台灣地方政府標準的施政總報告，適用於縣市長向縣市議會進行施政報告。

## 施政總報告定位

施政總報告是縣市長依據《地方制度法》規定，向縣市議會報告施政方針及施政成果的正式文件。報告需涵蓋：
- 過去會期的施政成果
- 當前施政重點與進度
- 未來施政方向與計畫
- 預算執行與財政狀況

## 標準架構

### 一、前言（約佔5-10%篇幅）

**目的**：開場致詞，說明施政背景與核心理念

**內容要素**：
- 議長、副議長及各位議員致敬
- 報告會期與時間範圍說明
- 施政願景與核心價值
- 當前經社情勢概述

**範例開頭**：
```
主席、副主席、各位議員女士、先生：

欣逢貴會第○屆第○次定期大會開議，○○（本人/春米）謹代表縣府團隊，
向大會提出施政總報告，敬請指教。

過去○年，本府秉持「○○○○」的施政理念，在議會的支持與監督下，
積極推動各項縣政建設...
```

---

### 二、施政成果回顧（約佔30-40%篇幅）

**目的**：展現過去施政績效，以具體數據佐證

**編排方式**：依施政領域分章節

**各領域應包含**：
1. **政策目標**：該領域的施政目標說明
2. **執行措施**：具體推動的政策與計畫
3. **量化成果**：以數據呈現績效
4. **質化成效**：市民/縣民實際感受的改變
5. **獲獎與肯定**：外部評比、獎項

**數據呈現格式**：
```
【交通建設】
- 完成道路拓寬：○○公里
- 新增停車位：○,○○○格
- 公車路線優化：新增○線、調整○線
- 滿意度提升：由○○%提升至○○%
```

---

### 三、分領域施政報告（約佔40-50%篇幅）

依縣市特性規劃施政主軸，以屏東縣為例的五大主軸：

#### 3.1 【韌性治理】- 防災、基礎建設、環境永續
- 災害防救機制
- 公共建設推動
- 環境保護措施
- 氣候調適策略

#### 3.2 【全齡照顧】- 社會福利、教育、醫療衛生
- 長照服務布建
- 托育及幼教政策
- 醫療資源配置
- 弱勢關懷措施

#### 3.3 【觀光發展】- 觀光、文化、城市行銷
- 觀光景點開發
- 文化資產保存
- 地方特色行銷
- 國際交流合作

#### 3.4 【產業經濟】- 產業、農業、就業
- 產業輔導與招商
- 農漁業現代化
- 青年就業促進
- 創新創業支援

#### 3.5 【安心生活】- 治安、交通、民政
- 治安維護強化
- 交通安全改善
- 戶政便民服務
- 鄰里社區營造

---

### 四、預算執行與財政報告（約佔5-10%篇幅）

**必要內容**：
- 年度預算執行率
- 重大建設經費配置
- 自籌財源與補助款比例
- 債務管理狀況

**格式範例**：
```
【○○○年度預算執行概況】

總預算：新台幣○○○億元
- 經常門：○○○億元（執行率○○%）
- 資本門：○○○億元（執行率○○%）

重點計畫經費：
1. ○○建設計畫：○億元
2. ○○推動方案：○億元
...
```

### 五、未來展望（約佔10-15%篇幅）

**內容要素**：
- 下一會期/年度施政重點
- 重大建設推動期程
- 政策目標與 KPI
- 資源配置規劃

**呈現方式**：
```
【未來施政重點】

一、短期目標（○個月內）
   1. ○○○○○
   2. ○○○○○

二、中期規劃（○-○年）
   1. ○○○○○
   2. ○○○○○

三、長期願景
   ○○○○○○○○○○
```

---

### 六、結語（約佔3-5%篇幅）

**內容要素**：
- 感謝議會支持與監督
- 重申施政決心
- 期勉未來合作
- 祈請指教

**範例結語**：
```
各位議員先生、女士：

縣政推動需要府會攜手合作，感謝議會長期以來的支持與鞭策。
○○將持續秉持○○○的精神，帶領縣府團隊全力以赴，
為○○縣民打造更美好的生活環境。

以上施政總報告，敬請各位議員不吝指教。謝謝！
```

## 撰寫風格指南

### 語言特色
- **正式書面語**：避免口語化表達
- **謙遜態度**：「本府」、「本人」等謙稱
- **積極正面**：聚焦成果與展望
- **具體明確**：以數據佐證，避免空泛

### 常用詞彙

| 場景 | 推薦用詞 |
|------|---------|
| 開場 | 欣逢、謹代表、敬請指教 |
| 成果 | 推動、落實、完成、達成、提升 |
| 數據 | 較去年成長○%、累計達○○ |
| 感謝 | 感謝議會鼎力支持、在議會的監督與協助下 |
| 未來 | 將持續、規劃推動、積極爭取 |

### 避免用詞
- 過於口語化的表達
- 負面或抱怨性質的內容
- 攻擊性或批評其他單位的言詞
- 未經證實的數據或承諾

## 實務撰寫流程

### 步驟一：資料蒐集
- 各局處業務報告
- 年度施政計畫執行情形
- 預算執行報告
- 重大政策推動成果
- 民調與滿意度調查

### 步驟二：架構規劃
- 確認報告會期與範圍
- 規劃施政主軸（建議3-5個）
- 分配各章節篇幅比例

### 步驟三：內容撰寫
- 由各局處提供素材
- 研考單位統整編輯
- 秘書處潤稿定稿

### 步驟四：審核流程
- 各局處首長確認
- 秘書長審閱
- 縣市長核定

## 附錄：相關法規依據

### 地方制度法相關條文
- **第48條**：直轄市議會、縣（市）議會、鄉（鎮、市）民代表會定期會開會時，直轄市長、縣（市）長、鄉（鎮、市）長應提出施政報告。
- **第49條**：直轄市議會、縣（市）議會、鄉（鎮、市）民代表會得邀請首長列席報告。

### 預算法相關條文
- **第46條**：各主管機關應將其年度施政計畫併同歲入、歲出概算，提送行政院核定。
