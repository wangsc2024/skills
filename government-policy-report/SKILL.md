# 台灣地方政府施政總報告撰寫指南

本技能協助撰寫符合台灣地方政府標準的施政總報告，適用於縣市長向縣市議會進行施政報告。

## 施政總報告定位

施政總報告是縣市長依據《地方制度法》規定，向縣市議會報告施政方針及施政成果的正式文件。報告需涵蓋：
- 過去會期的施政成果
- 當前施政重點與進度
- 未來施政方向與計畫
- 預算執行與財政狀況

## 標準架構

### 一、前言（約佔5-10%篇幅）

**目的**：開場致詞，說明施政背景與核心理念

**內容要素**：
- 議長、副議長及各位議員致敬
- 報告會期與時間範圍說明
- 施政願景與核心價值
- 當前經社情勢概述

**範例開頭**：
```
主席、副主席、各位議員女士、先生：

欣逢貴會第○屆第○次定期大會開議，○○（本人/春米）謹代表縣府團隊，
向大會提出施政總報告，敬請指教。

過去○年，本府秉持「○○○○」的施政理念，在議會的支持與監督下，
積極推動各項縣政建設...
```

### 二、施政成果回顧（約佔30-40%篇幅）

**目的**：展現過去施政績效，以具體數據佐證

**編排方式**：依施政領域分章節

**各領域應包含**：
1. **政策目標**：該領域的施政目標說明
2. **執行措施**：具體推動的政策與計畫
3. **量化成果**：以數據呈現績效
4. **質化成效**：市民/縣民實際感受的改變
5. **獲獎與肯定**：外部評比、獎項

**數據呈現格式**：
```
【交通建設】
- 完成道路拓寬：○○公里
- 新增停車位：○,○○○格
- 公車路線優化：新增○線、調整○線
- 滿意度提升：由○○%提升至○○%
```

---

### 三、分領域施政報告（約佔40-50%篇幅）

依縣市特性規劃施政主軸，以屏東縣「五箭齊發」施政綱要為例：

#### 3.1 【屏東更韌性】- 防災、治水、基礎建設

**核心理念**：打造屏東成為更具韌性的希望城市

**重點施政**：
- **治水防災**：河川及區域排水改善工程
- **智慧防災**：防汛預警系統、防災資訊整合
- **供電穩定**：強化電網韌性、綠能發展
- **基礎建設**：道路拓寬、橋梁改建

**數據呈現範例**：
```
【治水建設】
- 113年度治水工程經費：新台幣15億4,759萬元
- 治水工程數量：15件
- 第八批工程核定經費：29億4,700萬元（10案）
- 114年應急工程：9,368萬元（8案）
```

#### 3.2 【全齡心照顧】- 社會福利、教育、醫療衛生

**核心理念**：從出生到終老的全方位照顧

**重點施政**：
- **生育支持**：生育津貼（第2胎以上提高至3萬元）
- **托育服務**：公共托育家園、托育補助
- **校園健康**：行動醫療入校園健檢、兒童發展篩檢
- **長照服務**：多層級樂活照顧服務園區、失智照護
- **行動醫療**：行動生活復能車（滿意度95%）、行動中醫照護

**數據呈現範例**：
```
【長照服務】
- 行動生活復能車：民眾滿意度95%
- 行動中醫照護：113年8月啟動
- 長照住宿式機構服務量能持續提升
- 引進屏東榮總、義大醫院等醫療資源
```

#### 3.3 【觀光新戰略】- 觀光、文化、城市行銷

**核心理念**：文化建設帶動城市翻轉

**重點施政**：
- **硬體建設**：風域半島、恆春文化中心民謠館、出火地質公園
- **港口升級**：後壁湖漁港、滿州港口吊橋、車城海口港
- **溫泉觀光**：四重溪溫泉公園
- **航運園區**：TAIWAN Hi 新園海洋航運園區
- **燈節活動**：屏東燈節（280萬人次）
- **漁港點亮**：7座漁港景觀照明計畫

**數據呈現範例**：
```
【觀光成果】
- 屏東燈節參觀人次：超過280萬人次
- 新園海洋航運園區：整合候船、停車、轉乘、觀景等多元服務
- 漁港點亮計畫：7座漁港
- 熱博、台灣祭、黑鮪魚觀光季：超過百萬人次
```

#### 3.4 【產業星未來】- 產業、農業、科技

**核心理念**：串連南台灣科技產業廊帶

**重點施政**：
- **高鐵特定區**：34年來首次新訂都市計畫，以高科技產業為主軸
- **科技園區**：屏東科學園區、屏東科技產業園區擴區
- **產業園區**：六塊厝產業園區、熱帶農業特色產業園區
- **農業升級**：農業物產館、農漁業現代化

**數據呈現範例**：
```
【產業發展】
- 高鐵屏東車站特定區：屏東縣34年來首次新訂都市計畫
- 預計創造就業機會：8,500個
- 預計年產值：約550億元
- 園區：屏東科學園區、科技產業園區、六塊厝產業園區
```

#### 3.5 【安心過生活】- 治安、交通、民政

**核心理念**：打造宜居城市，讓縣民安心生活

**重點施政**：
- **交通建設**：大型交通建設、道路安全改善
- **自來水**：提升自來水普及率
- **公共設施**：共融公園、圖書館、社會住宅
- **動物保護**：屏東動物之家改建
- **治安維護**：警察、消防、環保機關聯合執法

**數據呈現範例**：
```
【交通建設】
- 完成道路拓寬：屏187、屏187乙、屏200、核三周邊道路
- 生活圈道路建設持續推動
- 大型交通建設可行性研究進行中
```

## 撰寫風格指南

### 語言特色
- **正式書面語**：避免口語化表達
- **謙遜態度**：「本府」、「本人」等謙稱
- **積極正面**：聚焦成果與展望
- **具體明確**：以數據佐證，避免空泛

### 常用詞彙

| 場景 | 推薦用詞 |
|------|---------|
| 開場 | 欣逢、謹代表、敬請指教 |
| 成果 | 推動、落實、完成、達成、提升 |
| 數據 | 較去年成長○%、累計達○○ |
| 感謝 | 感謝議會鼎力支持、在議會的監督與協助下 |
| 未來 | 將持續、規劃推動、積極爭取 |

### 避免用詞
- 過於口語化的表達
- 負面或抱怨性質的內容
- 攻擊性或批評其他單位的言詞
- 未經證實的數據或承諾

## 屏東縣政府施政總報告專區

### 屏東縣施政總報告特色

屏東縣長周春米於113年（2024年）10月22日赴縣議會進行施政總報告，以「五箭齊發」為施政綱要，打造「希望大屏東」。

### 屏東縣「五箭齊發」架構

| 箭 | 主軸名稱 | 核心內涵 | 主要施政項目 |
|----|----------|----------|--------------|
| 第一箭 | 屏東更韌性 | 防災韌性城市 | 治水工程、智慧防災、供電穩定 |
| 第二箭 | 全齡心照顧 | 全人照護深化 | 生育津貼、長照服務、行動醫療 |
| 第三箭 | 觀光新戰略 | 文化帶動翻轉 | 燈節、漁港、觀光景點 |
| 第四箭 | 產業星未來 | 科技產業廊帶 | 高鐵特定區、科學園區 |
| 第五箭 | 安心過生活 | 宜居城市 | 交通、治安、公共設施 |

## 實務撰寫流程

### 步驟一：資料蒐集
- 各局處業務報告
- 年度施政計畫執行情形
- 預算執行報告
- 重大政策推動成果
- 民調與滿意度調查

### 步驟二：架構規劃
- 確認報告會期與範圍
- 規劃施政主軸（建議3-5個）
- 分配各章節篇幅比例

### 步驟三：內容撰寫
- 由各局處提供素材
- 研考單位統整編輯
- 秘書處潤稿定稿

### 步驟四：審核流程
- 各局處首長確認
- 秘書長審閱
- 縣市長核定

## 附錄：相關法規依據

### 地方制度法相關條文
- **第48條**：直轄市議會、縣（市）議會、鄉（鎮、市）民代表會定期會開會時，直轄市長、縣（市）長、鄉（鎮、市）長應提出施政報告。
- **第49條**：直轄市議會、縣（市）議會、鄉（鎮、市）民代表會得邀請首長列席報告。

### 預算法相關條文
- **第46條**：各主管機關應將其年度施政計畫併同歲入、歲出概算，提送行政院核定。
