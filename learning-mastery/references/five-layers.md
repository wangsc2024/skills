# 五層學習法詳解

## 框架概述

楊大輝的五層學習法（記、懂、網、拓、活）是一個循序漸進的學習框架，每一層都建立在前一層的基礎上。

## 第一層：記（Memory）

### 核心技巧

#### 提取練習 (Retrieval Practice)
- **定義**：主動從記憶中提取信息，而非被動重讀
- **原理**：每次提取都會強化記憶痕跡
- **方法**：
  - 閃卡練習
  - 自我測試
  - 回想筆記

#### 間隔複習 (Spaced Repetition)
- **定義**：在遺忘臨界點複習
- **原理**：對抗艾賓浩斯遺忘曲線
- **工具**：Anki、SuperMemo

#### 交錯練習 (Interleaving)
- **定義**：混合練習不同主題
- **原理**：增加區辨能力
- **應用**：ABCA 模式 vs AAAA 模式

### 常見錯誤
- 流暢性錯覺：誤以為熟悉等於記住
- 過度依賴重讀
- 忽略間隔複習的重要性

## 第二層：懂（Understanding）

### 核心技巧

#### 費曼技巧
1. 選擇一個概念
2. 用簡單語言解釋（給5歲小孩聽）
3. 發現解釋困難的地方
4. 回去重新學習
5. 簡化並使用類比

#### 理解的層次
- Level 1：能複述定義
- Level 2：能舉例說明
- Level 3：能預測結果
- Level 4：能解釋例外
- Level 5：能教會他人

### 檢驗理解的問題
- 「這是什麼？」
- 「為什麼會這樣？」
- 「如果...會怎樣？」
- 「這和...有什麼不同？」

## 第三層：網（Network）

### 核心技巧

#### 知識連結
- 因果連結：A 導致 B
- 類比連結：A 類似 B
- 對比連結：A 不同於 B
- 組成連結：A 是 B 的一部分

#### 心智模型
- 帕累托法則 (80/20)
- 第一性原理
- 反向思考
- 複利效應
- 機會成本

#### 概念地圖
- 視覺化知識結構
- 展示概念間關係
- 發現知識缺口

### 工具推薦
- Obsidian（雙向連結）
- Roam Research
- XMind

## 第四層：拓（Expansion）

### 橫向拓展
- **類比遷移**：將知識應用到其他領域
- **跨領域學習**：從不同領域獲取靈感
- **模式識別**：找出通用原則

### 縱向拓展
- **深度探索**：追溯歷史、研究前沿
- **專家知識**：學習專家如何思考
- **原始資料**：閱讀原始研究

### 拓展資源層次
1. 入門教科書
2. 專業教科書
3. 綜述論文
4. 原始研究
5. 專家訪談

## 第五層：活（Application）

### 核心技巧

#### 生產效應
- 寫作輸出
- 教學他人
- 專案實作
- 創意應用

#### 刻意練習
- 具體目標
- 完全專注
- 即時反饋
- 舒適圈外

### 應用場景
- 解決實際問題
- 創新和創造
- 教學和分享
- 專案和產品

## 層次間的關係

```
活 ← 需要拓的廣度和深度
↑
拓 ← 需要網的連結基礎
↑
網 ← 需要懂的理解支撐
↑
懂 ← 需要記的材料儲備
↑
記 ← 基礎層
```

每一層都是下一層的基礎，但學習過程並非線性，而是螺旋上升。
