# 學習技巧實踐指南

## 記憶技巧

### 提取練習實踐

#### 基礎方法
```
1. 學習內容（15分鐘）
2. 闔上書本
3. 寫下能記住的所有內容（5分鐘）
4. 打開書本驗證
5. 標記遺漏的部分
6. 重複練習遺漏部分
```

#### 進階方法：自我出題
```
閱讀一段內容後，問自己：
- 這段的主要論點是什麼？
- 有哪些關鍵概念？
- 作者舉了什麼例子？
- 這和前面學的有什麼關聯？
```

### 間隔複習實踐

#### Anki 使用策略
```
新卡片設置：
- 每日新卡片：20-30張
- 學習步伐：1m 10m（第一次錯誤後1分鐘，正確後10分鐘）
- 畢業間隔：1天

複習時：
- 選擇「再來一次」：完全不記得
- 選擇「困難」：勉強記得
- 選擇「良好」：正常記得
- 選擇「容易」：非常熟悉
```

#### 手動間隔複習
```
Day 1: 學習新內容
Day 2: 第一次複習
Day 4: 第二次複習
Day 7: 第三次複習
Day 14: 第四次複習
Day 30: 第五次複習
```

## 理解技巧

### 費曼技巧實踐

#### 範例：解釋「遞迴」

**Step 1：專業解釋**
「遞迴是一種程式設計技巧，函數會呼叫自身來解決問題。」

**Step 2：簡化版本**
「想像你在排隊，想知道自己是第幾個。
你問前面的人：『你是第幾個？』
他也不知道，就問他前面的人。
一直問到第一個人說：『我是第一個！』
然後答案往回傳：第二個、第三個...最後你知道自己是第幾個。
這就是遞迴：把大問題分成小問題，一直分到最簡單的問題。」

### 理解檢測問題

```
基礎理解：
- 這是什麼？（定義）
- 有什麼特點？（屬性）

中級理解：
- 可以舉個例子嗎？（例證）
- 為什麼是這樣？（原因）

高級理解：
- 如果改變X會怎樣？（預測）
- 這在什麼情況下不適用？（邊界）
- 這和Y有什麼關係？（連結）
```

## 連結技巧

### 概念地圖製作

#### 步驟
1. 確定核心概念（放在中間）
2. 列出相關概念
3. 用線條連接相關概念
4. 在線條上標註關係類型
5. 添加層次和分支

#### 關係類型標記
- → 導致
- ↔ 相互影響
- ⊃ 包含
- ≈ 類似
- ≠ 對比

### 雙向連結筆記

```markdown
# 提取練習

提取練習是一種主動學習方法。

相關連結：
- [[間隔複習]] - 配合使用效果更好
- [[測試效應]] - 理論基礎
- [[艾賓浩斯遺忘曲線]] - 解決的問題
- [[費曼技巧]] - 另一種主動學習方法
```

## 拓展技巧

### 類比思考練習

#### 範例：用烹飪類比程式設計
```
程式 = 食譜
函數 = 烹飪步驟
變數 = 食材
迴圈 = 重複攪拌
條件判斷 = 「如果太鹹就加水」
除錯 = 試吃調整
```

#### 類比練習步驟
1. 選擇源領域（熟悉的）
2. 選擇目標領域（想學的）
3. 找出結構相似性
4. 驗證類比是否成立
5. 發現類比的限制

### 深度閱讀策略

```
第一遍：瀏覽
- 看標題、摘要、結論
- 了解整體結構

第二遍：理解
- 仔細閱讀每個部分
- 標記不懂的地方

第三遍：質疑
- 作者的假設是什麼？
- 有什麼證據支持？
- 有什麼反對意見？

第四遍：連結
- 這和我知道的有什麼關聯？
- 可以應用在哪裡？
```

## 應用技巧

### 刻意練習設計

#### 練習規劃模板
```
目標：具體且可測量
├─ 現狀：當前能力水平
├─ 目標：想達到的水平
├─ 差距：需要提升的地方
└─ 練習計劃：
    ├─ 練習內容
    ├─ 練習時間
    ├─ 反饋方式
    └─ 調整機制
```

### 專案式學習

#### 流程
```
1. 選擇專案
   - 與學習目標相關
   - 難度適中（略超能力）

2. 規劃階段
   - 分解成小任務
   - 識別需要學習的知識

3. 執行階段
   - 邊做邊學
   - 記錄問題和解決方案

4. 反思階段
   - 什麼做得好？
   - 什麼可以改進？
   - 學到了什麼？
```

### 教學輸出

#### 教學準備
```
1. 確定教學目標
2. 整理知識結構
3. 準備例子和類比
4. 預想可能的問題
5. 設計練習題
```

#### 教學過程
```
導入（引起興趣）
↓
講解（核心概念）
↓
示範（實際例子）
↓
練習（學生操作）
↓
反饋（糾正錯誤）
↓
總結（重點回顧）
```

## 綜合練習計劃

### 每日學習流程
```
早晨（15分鐘）：
- Anki 複習

學習時段（90分鐘）：
- 25分鐘學習 + 5分鐘休息 × 3
- 每個番茄鐘結束進行提取練習

晚間（10分鐘）：
- 回顧今日學習
- 更新概念地圖
- 規劃明日複習
```

### 週學習計劃
```
週一至週五：
- 學習新內容
- 每日複習

週六：
- 週回顧
- 整理筆記
- 更新知識網絡

週日：
- 應用練習
- 專案工作
```
