# Vue DevTools Reference Index

## Quick Links
- [Official Documentation](https://devtools.vuejs.org/)
- [GitHub (v7)](https://github.com/vuejs/devtools)

## Core Features
- **Component Inspector** - View component tree and props
- **State Editing** - Modify data in real-time
- **Vuex/Pinia** - State management debugging
- **Time Travel** - Navigate state history
