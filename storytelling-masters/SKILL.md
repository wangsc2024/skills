# 說故事大師技藝 Skill

匯集商業溝通專家安奈特·西蒙斯、簡報設計大師乔西·杜乔爾特、神話學家乔瑟夫·坎貝爾，以及繪本作家大衛·卡利等人的說故事精華。

## 適用情境

當你需要：
- 運用故事影響和說服他人
- 設計令人難忘的商業簡報
- 建立與聽眾的情感連結
- 運用英雄旅程架構故事
- 用視覺思維說故事
- 建立信任並傳遞價值觀

## 核心大師與技法

### 1. 安奈特·西蒙斯 (Annette Simmons) - 影響力的六種故事

**核心理念：** 故事是最古老的影響工具，能以邏輯和數據無法達到的方式說服、激勵和啟發人心。

**六種必備故事：**

```
┌─────────────────────────────────────────────────────────┐
│                 影響力的六種故事                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  1. 「我是誰」的故事                                    │
│     → 建立可信度和真實性                               │
│                                                         │
│  2. 「我為何在這裡」的故事                              │
│     → 說明動機，消除防備                               │
│                                                         │
│  3. 「願景」故事                                        │
│     → 描繪未來，激發行動                               │
│                                                         │
│  4. 「教導」故事                                        │
│     → 傳遞知識和技能                                   │
│                                                         │
│  5. 「價值觀實踐」故事                                  │
│     → 展示而非說明價值觀                               │
│                                                         │
│  6. 「我知道你在想什麼」故事                            │
│     → 預先回應疑慮和反對                               │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**各類故事詳解：**

#### 1. 「我是誰」故事

**目的：** 讓聽眾認識你，建立信任

**結構：**
```
挑戰/困境 → 掙扎 → 領悟 → 改變
```

**示範：**
> "十年前，我是一個害怕公開演講的工程師。第一次報告時，我的聲音發抖，投影片順序全亂了。那次失敗讓我明白：技術再好，不會溝通就無法發揮影響力。從那天起，我開始學習說故事..."

**要點：**
- 展現真實和脆弱
- 包含具體細節
- 連結到你現在的專業

#### 2. 「我為何在這裡」故事

**目的：** 消除聽眾的防備心，說明你的動機

**關鍵問題：**
- 為什麼是你？
- 為什麼是現在？
- 你真正想要什麼？

**示範：**
> "我今天來不是要賣東西給你們。三年前，我目睹一個團隊因為溝通失敗而解散。如果當時有人教他們說故事的力量，結果會完全不同。這就是我在這裡的原因。"

#### 3. 「願景」故事

**目的：** 讓聽眾看見並渴望一個更好的未來

**技巧：**
- 用感官細節描繪未來
- 讓聽眾看到自己在這個未來中的角色
- 具體而非抽象

**示範：**
> "想像六個月後。你走進會議室，不再緊張。你講的故事讓客戶頻頻點頭，會議結束時，他們主動問：'我們什麼時候可以開始？'"

#### 4. 「教導」故事

**目的：** 透過故事傳遞知識，比直接說教更有效

**模式：**
```
問題情境 → 錯誤方法 → 正確方法 → 結果差異
```

**示範：**
> "有個銷售員總是先講產品規格。客戶禮貌地聽，然後說'我們考慮看看'。另一個銷售員先問客戶的挑戰，講了一個類似客戶的成功故事，然後才提產品。第二個銷售員的成交率是第一個的三倍。"

#### 5. 「價值觀實踐」故事

**目的：** 用具體事例展示抽象價值觀

**原則：**
- 不要說「我們重視誠信」
- 講一個體現誠信的真實故事

**示範：**
> "去年，一個客戶多付了我們五萬元，是他們的會計失誤。我們可以不說，但我們選擇退回那筆錢。客戶驚訝地說：'這種事我還是第一次遇到。'三個月後，他們的新專案指定要與我們合作。"

#### 6. 「我知道你在想什麼」故事

**目的：** 預先處理聽眾的疑慮和反對意見

**技巧：**
- 承認合理的懷疑
- 講述曾經有相同疑慮的人
- 展示他們如何轉變

**示範：**
> "你可能在想：'這對我不適用，我的行業不一樣。' 我曾經也這麼認為。直到我遇到一位核能工程師，他用故事說服監管機構批准了一個創新專案..."

### 2. 乔西·杜乔爾特 (Nancy Duarte) - 簡報說故事

**核心理念：** 偉大的簡報是旅程，帶領聽眾從現狀走向更好的未來。

**Duarte 方法的三元素：**

```
┌─────────────────────────────────────────┐
│         Duarte 方法核心                 │
├─────────────────────────────────────────┤
│                                         │
│    同理心 (Empathy)                     │
│        ↓                               │
│    故事 (Story)                        │
│        ↓                               │
│    視覺設計 (Visual Design)            │
│                                         │
└─────────────────────────────────────────┘
```

**簡報結構：現狀與未來的對比**

```
       ┌─────┐
       │未來 │     ╭──────╮
       │     │  ╭──╯      ╰──╮  ╭────╮
現狀 ──┤     ├──╯            ╰──╯    ╰──→ 新境界
       │     │
       └─────┘

圖示說明：
- 簡報應該在「現狀」和「可能的未來」之間擺動
- 每次對比都增加張力和渴望
- 最終將聽眾帶往新的境界
```

**關鍵技巧：**

1. **以聽眾為英雄**
   - 你不是英雄，你是導師
   - 聽眾是故事中的英雄
   - 你的角色是幫助他們成功

2. **製造對比**
   - 現狀 vs 可能的未來
   - 問題 vs 解決方案
   - 舊方法 vs 新方法

3. **S.T.A.R. 時刻**
   - **S**omething they'll always **R**emember
   - 創造令人難忘的瞬間
   - 可以是令人驚訝的統計、演示、或情感高峰

**TED 演講結構（Duarte 分析）：**

1. 開場：打破預期
2. 建立痛點
3. 提出解決方案
4. 展示證據
5. 呼籲行動

### 3. 乔瑟夫·坎貝爾 (Joseph Campbell) - 英雄旅程

**核心理念：** 所有偉大的故事都遵循相同的基本模式——英雄旅程。

**英雄旅程十二階段：**

```
                    ╭────────────────╮
                    │   特殊世界     │
                    │ (挑戰與轉變)   │
                ╭───┴────────────────┴───╮
              ╭─╯                         ╰─╮
            ╭─╯  考驗          獎賞         ╰─╮
          ╭─╯  3.進入         7.核心        ╰─╮
        ╭─╯   洞穴深處       考驗            ╰─╮
      ╭─╯                                     ╰─╮
    ╭─╯ 2.遇見            8.回歸之路           ╰─╮
  ╭─╯   導師                                    ╰─╮
╭─╯                                               ╰─╮
│ 1.啟程召喚                        9.復活/高潮    │
│                                                   │
│ 日常世界                          10.帶著靈藥   │
│ (平凡生活)                            歸來      │
╰───────────────────────────────────────────────────╯
```

**三幕結構簡化版：**

| 階段 | 內容 | 關鍵問題 |
|------|------|----------|
| **啟程** | 英雄離開熟悉的世界 | 是什麼驅使他行動？ |
| **歷練** | 英雄學習適應陌生世界 | 他面對什麼挑戰？ |
| **回歸** | 英雄帶著智慧回歸 | 他帶回什麼給眾人？ |

**商業應用：**

```
客戶的英雄旅程：

1. 日常世界 → 客戶目前的處境
2. 啟程召喚 → 他們遇到的問題
3. 拒絕召喚 → 為什麼他們還沒解決
4. 遇見導師 → 你/你的產品出現
5. 跨越門檻 → 決定嘗試
6. 考驗盟友 → 使用過程中的挑戰
7. 核心考驗 → 關鍵轉折點
8. 獎賞     → 獲得的成果
9. 歸返     → 新的日常
10. 靈藥    → 他們能分享的價值
```

### 4. 大衛·卡利 (Davide Calì) - 繪本敘事

**核心理念：** 用最少的文字和最精準的畫面，傳達深刻的情感和意義。

**卡利的技巧：**

1. **冷開場 (Cold Opening)**
   - 直接進入故事，不做過多鋪陳
   - 讓讀者從行動中理解情境

2. **視覺與文字的對話**
   - 文字說一件事，圖畫說另一件事
   - 兩者相加產生第三層意義

3. **留白的力量**
   - 不說明一切
   - 讓讀者/觀眾填補空白

4. **反轉與意外**
   - 挑戰讀者的預期
   - 用意外創造深刻印象

**《敵人》(The Enemy) 的敘事分析：**
- 使用對稱結構
- 兩個敵對士兵的視角交替
- 讀者同時理解兩方，而角色不知道
- 主題的傳達不靠說教，靠結構本身

## 說故事的通用原則

### 1. 情感連結優先

```
信息 → 遺忘
數據 → 遺忘
故事 → 記憶 + 行動
```

**為什麼故事有效？**
- 激活大腦多個區域
- 產生鏡像神經元反應
- 釋放催產素（信任荷爾蒙）

### 2. 具體勝過抽象

| 抽象 | 具體 |
|------|------|
| 我們幫助很多客戶 | 上個月，我們幫助了 47 家企業 |
| 品質很好 | 保固三年，故障率低於 0.1% |
| 他很有影響力 | 他一句話，三百人站起來鼓掌 |

### 3. 衝突是故事的引擎

**沒有衝突的故事：**
> "我們開發了一個產品，客戶喜歡，我們賺錢了。"

**有衝突的故事：**
> "我們開發了一個產品，第一批客戶退貨率 40%。我們幾乎放棄。但有一個客戶打電話來說...'

### 4. 脆弱建立連結

- 完美的人不值得信任
- 分享失敗和掙扎
- 展示你如何克服

### 5. 細節創造真實

**模糊：**
> "很多年前，在一個城市..."

**具體：**
> "2019 年三月，在台北信義區的一家星巴克..."

## 不同情境的故事模板

### 銷售場景

```
1. 「你可能遇過這種情況...」（建立共鳴）
2. 「我有個客戶，情況類似...」（引入故事）
3. 「他們嘗試了 X，結果...」（展示旅程）
4. 「現在他們...」（呈現轉變）
5. 「這讓我想到你...」（連結回聽眾）
```

### 領導力場景

```
1. 「我曾經犯過一個錯誤...」（建立可信度）
2. 「那次經驗教會我...」（分享教訓）
3. 「現在我們面對類似的情況...」（連結當下）
4. 「如果我們一起...」（展望未來）
```

### 演講場景

```
開場：令人意外的事實或場景
中段：三個支持論點，每個配一個故事
結尾：回到開場的元素，呼籲行動
```

## 快速參考

### 準備故事時

- [ ] 我想讓聽眾感受什麼？
- [ ] 我想讓他們做什麼？
- [ ] 這個故事有衝突嗎？
- [ ] 有具體的細節嗎？
- [ ] 結局有轉變嗎？

### 講述故事時

- 使用現在式增加臨場感
- 模仿對話中的語氣
- 適當停頓製造張力
- 觀察聽眾反應調整

### 故事後

- 不要解釋故事的意義
- 讓故事自己說話
- 連結到你的核心訊息
- 邀請行動

## 核心箴言

> **安奈特·西蒙斯：** "在你用數據說服別人之前，必須先用故事贏得他們的信任。"

> **乔西·杜乔爾特：** "偉大的簡報不是關於你，而是關於你的聽眾以及他們可以達到的地方。"

> **乔瑟夫·坎貝爾：** "我們進入洞穴時最害怕的地方，正是我們尋找的寶藏所在。"

## 注意事項

- 故事必須真實，虛假的故事會摧毀信任
- 故事是工具，不是操控手段
- 練習講述，直到自然流暢
- 最好的故事來自你自己的經歷
